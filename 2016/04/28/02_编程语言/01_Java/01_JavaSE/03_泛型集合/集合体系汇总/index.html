1. CollectionдҪ“зі»йӣҶеҗҲ

1.1 List иЎЁ

ListжҺҘеҸЈ
зү№зӮ№пјҡжңүеәҸгҖҒжңүдёӢж ҮгҖҒе…ғзҙ еҸҜйҮҚеӨҚ
иЎҘе……пјҡз»§жүҝдәҶзҲ¶жҺҘеҸЈжҸҗдҫӣзҡ„е…ұжҖ§ж–№жі•пјҢеҗҢж—¶е®ҡд№үдәҶдёҖдәӣзӢ¬жңүзҡ„дёҺдёӢж Үзӣёе…ізҡ„ж“ҚдҪңж–№жі•

Listе®һзҺ°зұ»пјҡArrayListгҖҒVectorгҖҒLinkedList

в‘  ArrayList зұ»пјҲж•°з»„пјү

зү№зӮ№пјҡ
1. ж•°з»„з»“жһ„е®һзҺ°пјҢжҹҘиҜўеҝ«гҖҒеўһеҲ ж…ўпјӣ
2. JDK1.2зүҲжң¬пјҢиҝҗиЎҢж•ҲзҺҮеҝ«гҖҒзәҝзЁӢдёҚе®үе…Ёпјӣ

еңәжҷҜпјҡжіЁеҶҢпјҲ1ж¬Ўпјү -> жҹҘиҜўпјҲnж¬Ўпјү
жіЁж„Ҹпјҡ

  • JDK7д№ӢеүҚпјҢж— еҸӮжһ„йҖ ж–№жі•е®һйҷ…еҲӣе»әй•ҝеәҰдёә 10 зҡ„Objectж•°з»„пјҢз”ЁиҝҳжҳҜдёҚз”ЁпјҢж•°з»„е°ұеңЁйӮЈйҮҢпјҢзҲұз”ЁдёҚз”Ё(еҚ дәҶеҶ…еӯҳ)
  • JDK8д№ӢеҗҺпјҢж— еҸӮжһ„йҖ ж–№жі•е®һйҷ…еҲӣе»әй•ҝеәҰдёә 0 зҡ„Objectж•°з»„пјҢйҰ–ж¬Ўaddе…ғзҙ ж—¶пјҢжүҚжү§иЎҢж•°з»„жү©е®№ж“ҚдҪңпјҢз„¶еҗҺзңҹжӯЈеҗ‘ж•°з»„дёӯжҸ’е…Ҙж•°жҚ®пјҲLazyжҮ’пјүпјҢз”Ёзҡ„ж—¶еҖҷеҲӣе»әжҲ–еҠ иҪҪпјҢжңүж•ҲйҷҚдҪҺж— з”ЁеҶ…еӯҳзҡ„еҚ з”ЁгҖӮ
в‘Ў Vector зұ»пјҲж•°з»„гҖҒзәҝзЁӢеҗҢжӯҘпјү

зү№зӮ№пјҡ
1. ж•°з»„з»“жһ„е®һзҺ°пјҢжҹҘиҜўеҝ«гҖҒеўһеҲ ж…ўпјӣ
2. JDK1.0зүҲжң¬пјҢиҝҗиЎҢж•ҲзҺҮж…ўгҖҒзәҝзЁӢе®үе…ЁгҖӮ

в‘ў LinkedList зұ»пјҲй“ҫиЎЁпјү

зү№зӮ№пјҡ
1. й“ҫиЎЁз»“жһ„е®һзҺ°пјҢеўһеҲ еҝ«пјҢжҹҘиҜўж…ўпјӣ

в‘Ј CopyOnWriteArrayList зұ»пјҲзәҝзЁӢе®үе…Ёдё”й«ҳж•Ҳзҡ„Listпјү

public class CopyOnWriteArrayList<E>
extends Object
implements List<E>, RandomAccess, Cloneable, Serializable
иҜҙжҳҺпјҡ

  • з¬ҰеҗҲListзү№зӮ№пјҡжңүеәҸгҖҒжңүдёӢж ҮгҖҒе…ғзҙ еҸҜйҮҚеӨҚ
  • зәҝзЁӢе®үе…Ёзҡ„ArrayListпјҢеҠ ејәзүҲиҜ»еҶҷеҲҶзҰ»пјӣ
  • еҶҷжңүй”ҒпјҢиҜ»ж— й”ҒпјҢиҜ»еҶҷд№Ӣй—ҙдёҚйҳ»еЎһпјҢдјҳдәҺиҜ»еҶҷй”Ғпјӣ
  • еҶҷе…Ҙж—¶пјҢе…ҲcopyдёҖдёӘе®№еҷЁеүҜжң¬гҖҒеҶҚж·»еҠ ж–°е…ғзҙ пјҢжңҖеҗҺжӣҝжҚўеј•з”Ёпјӣ
  • дҪҝз”Ёж–№ејҸдёҺArrayListж— ејӮгҖӮ

Listеҹәжң¬APIдҪҝз”ЁпјҡгҖҗJavaгҖ‘йӣҶеҗҲжЎҶжһ¶д№ӢListжҺҘеҸЈе’Ңе…¶е®һзҺ°зұ»

1.2 Set йӣҶеҗҲ

SetжҺҘеҸЈдёҺе®һзҺ°зұ»
зү№зӮ№пјҡ
1пјүж— еәҸгҖҒж— дёӢж ҮгҖҒе…ғзҙ дёҚеҸҜйҮҚеӨҚпјҲеҪ“жҸ’е…Ҙж–°е…ғзҙ ж—¶пјҢеҰӮжһңж–°е…ғзҙ дёҺе·Іжңүе…ғзҙ иҝӣиЎҢequalsжҜ”иҫғпјҢз»“жһңдёәtrueж—¶пјҢеҲҷжӢ’з»қж–°е…ғзҙ жҸ’е…Ҙпјү
2пјүsetжҺҘеҸЈе№¶жІЎжңүжҸҗдҫӣиҮӘе·ұзӢ¬жңүзҡ„ж–№жі•пјҢеқҮжҳҜз»§жүҝCollectionзҡ„ж–№жі•

Setе®һзҺ°зұ»пјҡHashSetгҖҒLinkedHashSetгҖҒTreeSet

в‘  HashSet зұ»пјҲHashCodeпјү

зү№зӮ№пјҡ

  • еҹәдәҺHashCodeе®һзҺ°е…ғзҙ дёҚйҮҚеӨҚ - ж— еәҸ
  • еҪ“еӯҳе…Ҙе…ғзҙ зҡ„е“ҲеёҢз ҒзӣёеҗҢж—¶пјҢдјҡи°ғз”ЁequalsзЎ®и®ӨпјҢз»“жһңдёәtrueпјҢеҲҷжӢ’з»қеҗҺиҖ…еҠ е…Ҙ
  • ж— еҸӮжһ„е»әеҲқе§Ӣе®№йҮҸдёә16пјҲиҙҹиҪҪеӣ еӯҗ0.75пјҢеҚі+75%е®№йҮҸжү©е®№пјү
  • еә•еұӮдҪҝз”Ёзҡ„HashMapзұ»пјҢеҚіе°ҶжүҖжңүйңҖиҰҒеӯҳеӮЁзҡ„еҖјпјҢйҖҡиҝҮHashMapеҺ»йҮҚеӯҳе…Ҙ
  • е…ҲеҲӨж–ӯhashCodeжҳҜеҗҰзӣёеҗҢпјҢеҶҚ==жҜ”иҫғең°еқҖжҳҜеҗҰзӣёеҗҢпјҢеҶҚequalsеҶ…е®№жҳҜеҗҰзӣёеҗҢ
в‘Ў LinkedHashSet зұ»пјҲи®°еҪ•жҸ’е…ҘйЎәеәҸпјү

зү№зӮ№пјҡ

  • з»§жүҝиҮӘHashSetпјҢеҸҲеҹәдәҺLinkedHashMapжқҘе®һзҺ°зҡ„
  • еә•еұӮдҪҝз”ЁLinkedHashMapпјҲй“ҫиЎЁз»“жһ„пјүеӯҳеӮЁпјҢиҠӮзӮ№еҪўејҸзӢ¬з«ӢеӯҳеӮЁж•°жҚ®пјҢе№¶еҸҜд»ҘжҢҮеҗ‘дёӢдёҖдёӘиҠӮзӮ№пјҢйҖҡиҝҮйЎәеәҸи®ҝй—®иҠӮзӮ№пјҢеҸҜдҝқз•ҷе…ғзҙ зҡ„жҸ’е…ҘйЎәеәҸ - жҸ’е…ҘйЎәеәҸ
  • жүҖжңүж–№жі•дёҺHashSetзӣёеҗҢпјҢз”Ёжі•д№ҹдёҖжЁЎдёҖж ·
в‘ў TreeSet зұ»пјҲдәҢеҸүж ‘-иҮӘеҠЁжҺ’еәҸпјү

зү№зӮ№пјҡ

  • еҹәдәҺжҺ’еҲ—йЎәеәҸе®һзҺ°е…ғзҙ дёҚйҮҚеӨҚ - иҮӘеҠЁжҺ’еәҸ
  • е®һзҺ°дәҶSortedSetжҺҘеҸЈпјҢеҜ№жүҖжңүжҸ’е…ҘйӣҶеҗҲзҡ„е…ғзҙ иҮӘеҠЁжҺ’еәҸ
  • е…ғзҙ еҜ№иұЎзҡ„зұ»еһӢеҝ…йЎ»е®һзҺ°ComparableжҺҘеҸЈпјҢжҢҮе®ҡжҺ’еәҸи§„еҲҷпјҲInteger/Stringзұ»й»ҳи®Өе®һзҺ°пјүпјҢйҖҡиҝҮйҮҚеҶҷCompareToж–№жі•жүҚиғҪдҪҝз”ЁпјҢд»ҘзЎ®е®ҡжҳҜеҗҰдёәйҮҚеӨҚе…ғзҙ 
в‘Ј CopyOnWriteArraySet зұ»пјҲзәҝзЁӢе®үе…Ёдё”й«ҳж•Ҳзҡ„Setпјү

public class CopyOnWriteArraySet<E>
extends AbstractSet<E>
implements Serializable
иҜҙжҳҺпјҡ

  • з¬ҰеҗҲSetзү№зӮ№пјҡж— еәҸгҖҒж— дёӢж ҮгҖҒе…ғзҙ дёҚйҮҚеӨҚ
  • зәҝзЁӢе®үе…Ёзҡ„SetпјҢеә•еұӮдҪҝз”ЁCopyOnWriteArrayListе®һзҺ°пјӣ
  • е”ҜдёҖдёҚеҗҢеңЁдәҺпјҢдҪҝз”ЁaddIfAbsent()ж·»еҠ е…ғзҙ (жҹҘйҮҚ)пјҢдјҡйҒҚеҺҶж•°з»„пјӣ
  • еҰӮеӯҳеңЁе…ғзҙ пјҢеҲҷдёҚж·»еҠ пјҲжү”жҺүеүҜжң¬пјүпјӣ
  • дҪҝз”Ёж–№ејҸдёҺHashSetж— ејӮгҖӮ

Setеҹәжң¬APIдҪҝз”ЁпјҡгҖҗJavaгҖ‘йӣҶеҗҲжЎҶжһ¶д№ӢSetжҺҘеҸЈгҖҒе®һзҺ°зұ»гҖҒжҺ’еәҸ

>> HashSetжҺ’еәҸзҡ„дёӨз§Қж–№жі•

1пјүйҒҚеҺҶеҠ е…ҘеҲ°ListдёӯдҪҝз”ЁCollections.sort(list)жҺ’еәҸпјӣ
2пјүдҪҝз”ЁTreeSetзҡ„жһ„йҖ еҲӣе»әдёҖдёӘTreeSetеҜ№иұЎе®һзҺ°иҮӘеҠЁжҺ’еәҸгҖӮ

1.3 Queue йҳҹеҲ—

  • Collectionзҡ„еӯҗжҺҘеҸЈпјҢиЎЁзӨәйҳҹеҲ—FIFOпјҲFirst In First Outпјү
    в‘  ConcurrentLinkedQueue зұ»пјҲзәҝзЁӢе®үе…Ёдё”й«ҳж•Ҳзҡ„Queueпјү
  • зәҝзЁӢе®үе…ЁгҖҒеҸҜй«ҳж•ҲиҜ»еҶҷзҡ„йҳҹеҲ—пјҢй«ҳе№¶еҸ‘дёӢжҖ§иғҪжңҖеҘҪзҡ„йҳҹеҲ—пјӣ
  • ж— й”ҒгҖҒCASжҜ”иҫғдәӨжҚўз®—жі•пјҢдҝ®ж”№зҡ„ж–№жі•еҢ…еҗ«3дёӘж ёеҝғеҸӮж•°(V,E,N)пјӣ
  • VпјҡиҰҒжӣҙж–°зҡ„еҸҳйҮҸгҖҒEпјҡйў„жңҹеҖјгҖҒNпјҡж–°еҖј
  • еҸӘжңүеҪ“V==Eж—¶пјҢV=NпјӣеҗҰеҲҷиЎЁзӨәе·Іиў«жӣҙж–°иҝҮпјҢеҲҷеҸ–ж¶ҲеҪ“еүҚж“ҚдҪңгҖӮ
в‘Ў ArrayBlockingQueue зұ»пјҲжңүз•Ңйҳ»еЎһйҳҹеҲ— - BlockingQueueе®һзҺ°зұ»пјү
  • ж•°з»„з»“жһ„е®һзҺ°пјҢжңүз•ҢйҳҹеҲ—гҖӮжүӢе·Ҙеӣәе®ҡдёҠйҷҗ
в‘ў LinkedBlockingQueueзұ»пјҲж— з•Ңйҳ»еЎһйҳҹеҲ— - BlockingQueueе®һзҺ°зұ»пјү
  • й“ҫиЎЁз»“жһ„е®һзҺ°пјҢж— з•ҢйҳҹеҲ—гҖӮй»ҳи®ӨдёҠйҷҗInteger.MAX_VALUE

Queueеҹәжң¬APIдҪҝз”ЁпјҡгҖҗJavaгҖ‘QueueгҖҒBlockingQueueе’ҢйҳҹеҲ—е®һзҺ°з”ҹдә§иҖ…ж¶Ҳиҙ№иҖ…жЁЎејҸ

2. MapдҪ“зі»йӣҶеҗҲ


зү№зӮ№пјҡ

  • з”ЁдәҺеӯҳеӮЁд»»ж„Ҹй”®еҖјеҜ№пјҲKey-Valueпјү
  • й”®пјҡж— еәҸгҖҒж— дёӢж ҮгҖҒдёҚе…Ғи®ёйҮҚеӨҚпјҲе”ҜдёҖпјү
  • еҖјпјҡж— еәҸгҖҒж— дёӢж ҮгҖҒе…Ғи®ёйҮҚеӨҚ

    в‘  HashMap зұ»пјҲж•°з»„+й“ҫиЎЁ+зәўй»‘ж ‘пјү

    зү№зӮ№пјҡ
  • JDK1.2зүҲжң¬пјҢзәҝзЁӢдёҚе®үе…ЁпјҢиҝҗиЎҢж•ҲзҺҮеҝ«пјӣе…Ғи®ёз”ЁnullдҪңдёәkeyжҲ–жҳҜvalue
  • ж— еҸӮжһ„е»әеҲқе§Ӣе®№йҮҸдёә16пјҲиҙҹиҪҪеӣ еӯҗ0.75пјҢеҚі+75%е®№йҮҸжү©е®№пјү
  • HashMapз®—жі•пјҡжӢҝеҲ°д»»дҪ•дёҖдёӘеҜ№иұЎеҗҺпјҢйҖҡиҝҮhash(key)еҒҡиҝҗз®—пјҢkey>>>16(йҷӨ16)пјҢеҸӘеҸҜиғҪеҫ—еҲ°0-15д№Ӣй—ҙзҡ„дёҖдёӘж•°з»„пјҢдҪңдёәжҸ’е…Ҙж•°з»„зҡ„дёӢж ҮгҖӮ

в‘Ў LinkedHashMap зұ»пјҲи®°еҪ•жҸ’е…ҘйЎәеәҸпјү

зү№зӮ№пјҡHashMap зҡ„дёҖдёӘеӯҗзұ»пјҢдҝқеӯҳдәҶи®°еҪ•зҡ„жҸ’е…ҘйЎәеәҸпјҢд№ҹеҸҜеңЁжһ„йҖ ж—¶еёҰеҸӮж•°пјҢжҢүз…§и®ҝй—®ж¬ЎеәҸжҺ’еәҸгҖӮ

в‘ў Hashtable зұ»пјҲзәҝзЁӢе®үе…Ёпјү

зү№зӮ№пјҡJDK1.0зүҲжң¬пјҢзәҝзЁӢе®үе…ЁпјҢиҝҗиЎҢж•ҲзҺҮж…ўпјӣдёҚе…Ғи®ёnullдҪңдёәkeyжҲ–жҳҜvalue

в‘Ј Properties зұ»пјҲй…ҚзҪ®ж–Үд»¶иҜ»еҸ–пјү

зү№зӮ№пјҡHashtableзҡ„еӯҗзұ»пјҢиҰҒжұӮkeyе’ҢvalueйғҪжҳҜStringпјҢйҖҡеёёз”ЁдәҺй…ҚзҪ®ж–Үд»¶зҡ„иҜ»еҸ–гҖӮ
з”Ёжі•пјҡзұ»дјјHashMapпјҢдёҚйңҖиҰҒжҢҮе®ҡжіӣеһӢпјҢжҺҘеҸЈеҸӮйҳ…apiж–ҮжЎЈ

в‘Ө TreeMap зұ»пјҲиҮӘеҠЁжҺ’еәҸпјү

зү№зӮ№пјҡе®һзҺ°дәҶSortedMapжҺҘеҸЈпјҲMapзҡ„еӯҗжҺҘеҸЈпјүпјҢеҸҜд»ҘеҜ№keyиҮӘеҠЁжҺ’еәҸгҖӮ

в‘Ҙ ConcurrentHashMap зұ»пјҲзәҝзЁӢе®үе…Ёдё”й«ҳж•Ҳзҡ„Mapпјү

public class ConcurrentHashMap<K,V>
extends AbstractMap<K,V>
implements ConcurrentMap<K,V>, Serializable
иҜҙжҳҺпјҡ

  • еҲқиҜҶе®№йҮҸй»ҳи®Өдёә16ж®ө(Segment)пјҢдҪҝз”ЁеҲҶж®өй”Ғи®ҫи®Ўпјӣ
  • дёҚеҜ№ж•ҙдёӘMapеҠ й”ҒпјҢиҖҢжҳҜдёәжҜҸдёӘSegmentеҠ й”Ғпјӣ
  • еҪ“еӨҡдёӘеҜ№иұЎеӯҳе…ҘеҗҢдёҖдёӘSegmentж—¶пјҢжүҚйңҖиҰҒдә’ж–Ҙпјӣ
  • жңҖзҗҶжғізҠ¶жҖҒдҪҚ16дёӘеҜ№иұЎеҲҶеҲ«еӯҳе…Ҙ16дёӘSegmentпјҢе№¶иЎҢзәҝзЁӢж•°йҮҸ16дёӘпјӣ
  • дҪҝз”Ёж–№ејҸдёҺHashMapж— ејӮгҖӮ
    // JDK1.7: еҲҶж®өй”Ғи®ҫи®Ў Segment
    // JDK1.8: CASдәӨжҚўз®—жі•пјҲCASжҜ”иҫғе’ҢдәӨжҚўпјү + еҗҢжӯҘй”ҒпјҲй”Ғзҡ„жҳҜиЎЁеӨҙпјү

Mapеҹәжң¬APIдҪҝз”ЁпјҡгҖҗJavaгҖ‘йӣҶеҗҲжЎҶжһ¶д№ӢMapжҺҘеҸЈгҖҒе®һзҺ°зұ»иҜҰи§Ј
