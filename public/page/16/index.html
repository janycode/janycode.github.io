03-Git 协作开发

整体流程： Git 公共仓库账号 → 邀请协作伙伴 → 协作伙伴都同意。 Git 公共仓库账号 → 搭好项目架子上传 → Git链接 → 作为基础代码可以被 被邀请者 pull(拉) 到本地。 Git 公共仓库账号 → 创建 dev 分支 → push 远程仓库（即上传，此时 dev 分支代码同 master 代码） 协作者 → IDEA登陆 Git → 打开Git链接 → Fork到自己仓库
