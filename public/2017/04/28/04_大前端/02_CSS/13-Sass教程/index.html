13-Sass教程

参考资料：

  • sass官网：https://sass-lang.com.cn/
  • sass教程：https://sass-lang.com.cn/documentation/

1. 介绍

sass最成熟、最稳定、最强大的专业级css扩展语言

  • sass是css的预编译工具
  • 更优雅的书写css
  • 需要转换成css才能让浏览器认识和运行

安装一个插件：Easy Sass

  • 可以自动将.scss的样式代码文件转成 css 代码文件，即会自动生成 .css 和 .min.css 两个文件
