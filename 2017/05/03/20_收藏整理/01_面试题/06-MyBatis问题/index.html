MyBatisжҳҜд»Җд№Ҳпјҹ

MyBatis жҳҜдёҖж¬ҫдјҳз§Җзҡ„жҢҒд№…еұӮжЎҶжһ¶пјҢдёҖдёӘеҚҠ ORMпјҲеҜ№иұЎе…ізі»жҳ е°„пјүжЎҶжһ¶пјҢе®ғж”ҜжҢҒе®ҡеҲ¶еҢ– SQLгҖҒеӯҳеӮЁиҝҮзЁӢд»ҘеҸҠй«ҳзә§жҳ е°„гҖӮMyBatis йҒҝе…ҚдәҶеҮ д№ҺжүҖжңүзҡ„ JDBC д»Јз Ғе’ҢжүӢеҠЁи®ҫзҪ®еҸӮж•°д»ҘеҸҠиҺ·еҸ–з»“жһңйӣҶгҖӮMyBatis еҸҜд»ҘдҪҝз”Ёз®ҖеҚ•зҡ„ XML жҲ–жіЁи§ЈжқҘй…ҚзҪ®е’Ңжҳ е°„еҺҹз”ҹзұ»еһӢгҖҒжҺҘеҸЈе’Ң Java зҡ„ POJOпјҲPlain Old Java ObjectsпјҢжҷ®йҖҡиҖҒејҸ Java еҜ№иұЎпјүдёәж•°жҚ®еә“дёӯзҡ„и®°еҪ•гҖӮ

ORMжҳҜд»Җд№Ҳ

ORMпјҲObject Relational MappingпјүпјҢеҜ№иұЎе…ізі»жҳ е°„пјҢжҳҜдёҖз§ҚдёәдәҶи§ЈеҶіе…ізі»еһӢж•°жҚ®еә“ж•°жҚ®дёҺз®ҖеҚ•JavaеҜ№иұЎпјҲPOJOпјүзҡ„жҳ е°„е…ізі»зҡ„жҠҖжңҜгҖӮз®ҖеҚ•зҡ„иҜҙпјҢORMжҳҜйҖҡиҝҮдҪҝз”ЁжҸҸиҝ°еҜ№иұЎе’Ңж•°жҚ®еә“д№Ӣй—ҙжҳ е°„зҡ„е…ғж•°жҚ®пјҢе°ҶзЁӢеәҸдёӯзҡ„еҜ№иұЎиҮӘеҠЁжҢҒд№…еҢ–еҲ°е…ізі»еһӢж•°жҚ®еә“дёӯгҖӮ

дёәд»Җд№ҲиҜҙMybatisжҳҜеҚҠиҮӘеҠЁORMжҳ е°„е·Ҙе…·пјҹе®ғдёҺе…ЁиҮӘеҠЁзҡ„еҢәеҲ«еңЁе“ӘйҮҢпјҹ

HibernateеұһдәҺе…ЁиҮӘеҠЁORMжҳ е°„е·Ҙе…·пјҢдҪҝз”ЁHibernateжҹҘиҜўе…іиҒ”еҜ№иұЎжҲ–иҖ…е…іиҒ”йӣҶеҗҲеҜ№иұЎж—¶пјҢеҸҜд»Ҙж №жҚ®еҜ№иұЎе…ізі»жЁЎеһӢзӣҙжҺҘиҺ·еҸ–пјҢжүҖд»Ҙе®ғжҳҜе…ЁиҮӘеҠЁзҡ„гҖӮ

6пјүMappedStatement еҜ№иұЎпјҡеңЁ Executor жҺҘеҸЈзҡ„жү§иЎҢж–№жі•дёӯжңүдёҖдёӘ MappedStatement зұ»еһӢзҡ„еҸӮж•°пјҢиҜҘеҸӮж•°жҳҜеҜ№жҳ е°„дҝЎжҒҜзҡ„е°ҒиЈ…пјҢз”ЁдәҺеӯҳеӮЁиҰҒжҳ е°„зҡ„ SQL иҜӯеҸҘзҡ„ idгҖҒеҸӮж•°зӯүдҝЎжҒҜгҖӮ

7пјүиҫ“е…ҘеҸӮж•°жҳ е°„пјҡиҫ“е…ҘеҸӮж•°зұ»еһӢеҸҜд»ҘжҳҜ MapгҖҒList зӯүйӣҶеҗҲзұ»еһӢпјҢд№ҹеҸҜд»ҘжҳҜеҹәжң¬ж•°жҚ®зұ»еһӢе’Ң POJO зұ»еһӢгҖӮиҫ“е…ҘеҸӮж•°жҳ е°„иҝҮзЁӢзұ»дјјдәҺ JDBC еҜ№ preparedStatement еҜ№иұЎи®ҫзҪ®еҸӮж•°зҡ„иҝҮзЁӢгҖӮ

8пјүиҫ“еҮәз»“жһңжҳ е°„пјҡиҫ“еҮәз»“жһңзұ»еһӢеҸҜд»ҘжҳҜ MapгҖҒ List зӯүйӣҶеҗҲзұ»еһӢпјҢд№ҹеҸҜд»ҘжҳҜеҹәжң¬ж•°жҚ®зұ»еһӢе’Ң POJO зұ»еһӢгҖӮиҫ“еҮәз»“жһңжҳ е°„иҝҮзЁӢзұ»дјјдәҺ JDBC еҜ№з»“жһңйӣҶзҡ„и§ЈжһҗиҝҮзЁӢгҖӮ

MyBatisзҡ„еҠҹиғҪжһ¶жһ„жҳҜжҖҺж ·зҡ„

жҲ‘д»¬жҠҠMybatisзҡ„еҠҹиғҪжһ¶жһ„еҲҶдёәдёүеұӮпјҡ

  • APIжҺҘеҸЈеұӮпјҡжҸҗдҫӣз»ҷеӨ–йғЁдҪҝз”Ёзҡ„жҺҘеҸЈAPIпјҢејҖеҸ‘дәәе‘ҳйҖҡиҝҮиҝҷдәӣжң¬ең°APIжқҘж“Қзәөж•°жҚ®еә“гҖӮжҺҘеҸЈеұӮдёҖжҺҘж”¶еҲ°и°ғз”ЁиҜ·жұӮе°ұдјҡи°ғз”Ёж•°жҚ®еӨ„зҗҶеұӮжқҘе®ҢжҲҗе…·дҪ“зҡ„ж•°жҚ®еӨ„зҗҶгҖӮ
  • ж•°жҚ®еӨ„зҗҶеұӮпјҡиҙҹиҙЈе…·дҪ“зҡ„SQLжҹҘжүҫгҖҒSQLи§ЈжһҗгҖҒSQLжү§иЎҢе’Ңжү§иЎҢз»“жһңжҳ е°„еӨ„зҗҶзӯүгҖӮе®ғдё»иҰҒзҡ„зӣ®зҡ„жҳҜж №жҚ®и°ғз”Ёзҡ„иҜ·жұӮе®ҢжҲҗдёҖж¬Ўж•°жҚ®еә“ж“ҚдҪңгҖӮ
  • еҹәзЎҖж”Ҝж’‘еұӮпјҡиҙҹиҙЈжңҖеҹәзЎҖзҡ„еҠҹиғҪж”Ҝж’‘пјҢеҢ…жӢ¬иҝһжҺҘз®ЎзҗҶгҖҒдәӢеҠЎз®ЎзҗҶгҖҒй…ҚзҪ®еҠ иҪҪе’Ңзј“еӯҳеӨ„зҗҶпјҢиҝҷдәӣйғҪжҳҜе…ұз”Ёзҡ„дёңиҘҝпјҢе°Ҷд»–д»¬жҠҪеҸ–еҮәжқҘдҪңдёәжңҖеҹәзЎҖзҡ„з»„д»¶гҖӮдёәдёҠеұӮзҡ„ж•°жҚ®еӨ„зҗҶеұӮжҸҗдҫӣжңҖеҹәзЎҖзҡ„ж”Ҝж’‘гҖӮ

MyBatisзҡ„жЎҶжһ¶жһ¶жһ„и®ҫи®ЎжҳҜжҖҺд№Ҳж ·зҡ„

иҝҷеј еӣҫд»ҺдёҠеҫҖдёӢзңӢгҖӮMyBatisзҡ„еҲқе§ӢеҢ–пјҢдјҡд»Һmybatis-config.xmlй…ҚзҪ®ж–Үд»¶пјҢи§Јжһҗжһ„йҖ жҲҗConfigurationиҝҷдёӘзұ»пјҢе°ұжҳҜеӣҫдёӯзҡ„зәўжЎҶгҖӮ

(1)еҠ иҪҪй…ҚзҪ®пјҡй…ҚзҪ®жқҘжәҗдәҺдёӨдёӘең°ж–№пјҢдёҖеӨ„жҳҜй…ҚзҪ®ж–Үд»¶пјҢдёҖеӨ„жҳҜJavaд»Јз Ғзҡ„жіЁи§ЈпјҢе°ҶSQLзҡ„й…ҚзҪ®дҝЎжҒҜеҠ иҪҪжҲҗдёәдёҖдёӘдёӘMappedStatementеҜ№иұЎпјҲеҢ…жӢ¬дәҶдј е…ҘеҸӮж•°жҳ е°„й…ҚзҪ®гҖҒжү§иЎҢзҡ„SQLиҜӯеҸҘгҖҒз»“жһңжҳ е°„й…ҚзҪ®пјүпјҢеӯҳеӮЁеңЁеҶ…еӯҳдёӯгҖӮ

(2)SQLи§ЈжһҗпјҡеҪ“APIжҺҘеҸЈеұӮжҺҘж”¶еҲ°и°ғз”ЁиҜ·жұӮж—¶пјҢдјҡжҺҘж”¶еҲ°дј е…ҘSQLзҡ„IDе’Ңдј е…ҘеҜ№иұЎпјҲеҸҜд»ҘжҳҜMapгҖҒJavaBeanжҲ–иҖ…еҹәжң¬ж•°жҚ®зұ»еһӢпјүпјҢMybatisдјҡж №жҚ®SQLзҡ„IDжүҫеҲ°еҜ№еә”зҡ„MappedStatementпјҢз„¶еҗҺж №жҚ®дј е…ҘеҸӮж•°еҜ№иұЎеҜ№MappedStatementиҝӣиЎҢи§ЈжһҗпјҢи§ЈжһҗеҗҺеҸҜд»Ҙеҫ—еҲ°жңҖз»ҲиҰҒжү§иЎҢзҡ„SQLиҜӯеҸҘе’ҢеҸӮж•°гҖӮ

(3)SQLжү§иЎҢпјҡе°ҶжңҖз»Ҳеҫ—еҲ°зҡ„SQLе’ҢеҸӮж•°жӢҝеҲ°ж•°жҚ®еә“иҝӣиЎҢжү§иЎҢпјҢеҫ—еҲ°ж“ҚдҪңж•°жҚ®еә“зҡ„з»“жһңгҖӮ

(4)з»“жһңжҳ е°„пјҡе°Ҷж“ҚдҪңж•°жҚ®еә“зҡ„з»“жһңжҢүз…§жҳ е°„зҡ„й…ҚзҪ®иҝӣиЎҢиҪ¬жҚўпјҢеҸҜд»ҘиҪ¬жҚўжҲҗHashMapгҖҒJavaBeanжҲ–иҖ…еҹәжң¬ж•°жҚ®зұ»еһӢпјҢе№¶е°ҶжңҖз»Ҳз»“жһңиҝ”еӣһгҖӮ

дёәд»Җд№ҲйңҖиҰҒйў„зј–иҜ‘

  1. е®ҡд№үпјҡ
    SQL йў„зј–иҜ‘жҢҮзҡ„жҳҜж•°жҚ®еә“й©ұеҠЁеңЁеҸ‘йҖҒ SQL иҜӯеҸҘе’ҢеҸӮж•°з»ҷ DBMS д№ӢеүҚеҜ№ SQL иҜӯеҸҘиҝӣиЎҢзј–иҜ‘пјҢиҝҷж · DBMS жү§иЎҢ SQL ж—¶пјҢе°ұдёҚйңҖиҰҒйҮҚж–°зј–иҜ‘гҖӮ

  2. дёәд»Җд№ҲйңҖиҰҒйў„зј–иҜ‘
    JDBC дёӯдҪҝз”ЁеҜ№иұЎ PreparedStatement жқҘжҠҪиұЎйў„зј–иҜ‘иҜӯеҸҘпјҢдҪҝз”Ёйў„зј–иҜ‘гҖӮйў„зј–иҜ‘йҳ¶ж®өеҸҜд»ҘдјҳеҢ– SQL зҡ„жү§иЎҢгҖӮйў„зј–иҜ‘д№ӢеҗҺзҡ„ SQL еӨҡж•°жғ…еҶөдёӢеҸҜд»ҘзӣҙжҺҘжү§иЎҢпјҢDBMS дёҚйңҖиҰҒеҶҚж¬Ўзј–иҜ‘пјҢи¶ҠеӨҚжқӮзҡ„SQLпјҢзј–иҜ‘зҡ„еӨҚжқӮеәҰе°Ҷи¶ҠеӨ§пјҢйў„зј–иҜ‘йҳ¶ж®өеҸҜд»ҘеҗҲе№¶еӨҡж¬Ўж“ҚдҪңдёәдёҖдёӘж“ҚдҪңгҖӮеҗҢж—¶йў„зј–иҜ‘иҜӯеҸҘеҜ№иұЎеҸҜд»ҘйҮҚеӨҚеҲ©з”ЁгҖӮжҠҠдёҖдёӘ SQL йў„зј–иҜ‘еҗҺдә§з”ҹзҡ„ PreparedStatement еҜ№иұЎзј“еӯҳдёӢжқҘпјҢдёӢж¬ЎеҜ№дәҺеҗҢдёҖдёӘSQLпјҢеҸҜд»ҘзӣҙжҺҘдҪҝз”ЁиҝҷдёӘзј“еӯҳзҡ„ PreparedState еҜ№иұЎгҖӮMybatisй»ҳи®Өжғ…еҶөдёӢпјҢе°ҶеҜ№жүҖжңүзҡ„ SQL иҝӣиЎҢйў„зј–иҜ‘гҖӮ

MybatisйғҪжңүе“ӘдәӣExecutorжү§иЎҢеҷЁпјҹе®ғд»¬д№Ӣй—ҙзҡ„еҢәеҲ«жҳҜд»Җд№Ҳпјҹ

Mybatisжңүдёүз§Қеҹәжң¬зҡ„Executorжү§иЎҢеҷЁ: SimpleExecutorгҖҒReuseExecutorгҖҒBatchExecutorгҖӮ

SimpleExecutorпјҡжҜҸжү§иЎҢдёҖж¬ЎupdateжҲ–selectпјҢе°ұејҖеҗҜдёҖдёӘStatementеҜ№иұЎпјҢз”Ёе®Ңз«ӢеҲ»е…ій—ӯStatementеҜ№иұЎгҖӮ

ReuseExecutorпјҡжү§иЎҢupdateжҲ–selectпјҢд»ҘsqlдҪңдёәkeyжҹҘжүҫStatementеҜ№иұЎпјҢеӯҳеңЁе°ұдҪҝз”ЁпјҢдёҚеӯҳеңЁе°ұеҲӣе»әпјҢз”Ёе®ҢеҗҺпјҢдёҚе…ій—ӯStatementеҜ№иұЎпјҢиҖҢжҳҜж”ҫзҪ®дәҺMap<String, Statement>еҶ…пјҢдҫӣдёӢдёҖж¬ЎдҪҝз”ЁгҖӮз®ҖиЁҖд№ӢпјҢе°ұжҳҜйҮҚеӨҚдҪҝз”ЁStatementеҜ№иұЎгҖӮ

BatchExecutorпјҡжү§иЎҢupdateпјҲжІЎжңүselectпјҢJDBCжү№еӨ„зҗҶдёҚж”ҜжҢҒselectпјүпјҢе°ҶжүҖжңүsqlйғҪж·»еҠ еҲ°жү№еӨ„зҗҶдёӯпјҲaddBatch()пјүпјҢзӯүеҫ…з»ҹдёҖжү§иЎҢпјҲexecuteBatch()пјүпјҢе®ғзј“еӯҳдәҶеӨҡдёӘStatementеҜ№иұЎпјҢжҜҸдёӘStatementеҜ№иұЎйғҪжҳҜaddBatch()е®ҢжҜ•еҗҺпјҢзӯүеҫ…йҖҗдёҖжү§иЎҢexecuteBatch()жү№еӨ„зҗҶгҖӮдёҺJDBCжү№еӨ„зҗҶзӣёеҗҢгҖӮ

дҪңз”ЁиҢғеӣҙпјҡExecutorзҡ„иҝҷдәӣзү№зӮ№пјҢйғҪдёҘж јйҷҗеҲ¶еңЁSqlSessionз”ҹе‘Ҫе‘ЁжңҹиҢғеӣҙеҶ…гҖӮ

MybatisдёӯеҰӮдҪ•жҢҮе®ҡдҪҝз”Ёе“ӘдёҖз§ҚExecutorжү§иЎҢеҷЁпјҹ

еңЁMybatisй…ҚзҪ®ж–Үд»¶дёӯпјҢеңЁи®ҫзҪ®пјҲsettingsпјүеҸҜд»ҘжҢҮе®ҡй»ҳи®Өзҡ„ExecutorTypeжү§иЎҢеҷЁзұ»еһӢпјҢд№ҹеҸҜд»ҘжүӢеҠЁз»ҷDefaultSqlSessionFactoryзҡ„еҲӣе»әSqlSessionзҡ„ж–№жі•дј йҖ’ExecutorTypeзұ»еһӢеҸӮж•°пјҢеҰӮSqlSession openSession(ExecutorType execType)гҖӮ

й…ҚзҪ®й»ҳи®Өзҡ„жү§иЎҢеҷЁгҖӮSIMPLE е°ұжҳҜжҷ®йҖҡзҡ„жү§иЎҢеҷЁпјӣREUSE жү§иЎҢеҷЁдјҡйҮҚз”Ёйў„еӨ„зҗҶиҜӯеҸҘпјҲprepared statementsпјүпјӣ BATCH жү§иЎҢеҷЁе°ҶйҮҚз”ЁиҜӯеҸҘе№¶жү§иЎҢжү№йҮҸжӣҙж–°гҖӮ

MybatisжҳҜеҗҰж”ҜжҢҒе»¶иҝҹеҠ иҪҪпјҹеҰӮжһңж”ҜжҢҒпјҢе®ғзҡ„е®һзҺ°еҺҹзҗҶжҳҜд»Җд№Ҳпјҹ

Mybatisд»…ж”ҜжҢҒassociationе…іиҒ”еҜ№иұЎе’Ңcollectionе…іиҒ”йӣҶеҗҲеҜ№иұЎзҡ„е»¶иҝҹеҠ иҪҪпјҢassociationжҢҮзҡ„е°ұжҳҜдёҖеҜ№дёҖпјҢcollectionжҢҮзҡ„е°ұжҳҜдёҖеҜ№еӨҡжҹҘиҜўгҖӮеңЁMybatisй…ҚзҪ®ж–Үд»¶дёӯпјҢеҸҜд»Ҙй…ҚзҪ®жҳҜеҗҰеҗҜз”Ёе»¶иҝҹеҠ иҪҪlazyLoadingEnabled=true|falseгҖӮ

е®ғзҡ„еҺҹзҗҶжҳҜпјҢдҪҝз”ЁCGLIBеҲӣе»әзӣ®ж ҮеҜ№иұЎзҡ„д»ЈзҗҶеҜ№иұЎпјҢеҪ“и°ғз”Ёзӣ®ж Үж–№жі•ж—¶пјҢиҝӣе…ҘжӢҰжҲӘеҷЁж–№жі•пјҢжҜ”еҰӮи°ғз”Ёa.getB().getName()пјҢжӢҰжҲӘеҷЁinvoke()ж–№жі•еҸ‘зҺ°a.getB()жҳҜnullеҖјпјҢйӮЈд№Ҳе°ұдјҡеҚ•зӢ¬еҸ‘йҖҒдәӢе…ҲдҝқеӯҳеҘҪзҡ„жҹҘиҜўе…іиҒ”BеҜ№иұЎзҡ„sqlпјҢжҠҠBжҹҘиҜўдёҠжқҘпјҢз„¶еҗҺи°ғз”Ёa.setB(b)пјҢдәҺжҳҜaзҡ„еҜ№иұЎbеұһжҖ§е°ұжңүеҖјдәҶпјҢжҺҘзқҖе®ҢжҲҗa.getB().getName()ж–№жі•зҡ„и°ғз”ЁгҖӮиҝҷе°ұжҳҜе»¶иҝҹеҠ иҪҪзҡ„еҹәжң¬еҺҹзҗҶгҖӮ

еҪ“з„¶дәҶпјҢдёҚе…үжҳҜMybatisпјҢеҮ д№ҺжүҖжңүзҡ„еҢ…жӢ¬HibernateпјҢж”ҜжҢҒе»¶иҝҹеҠ иҪҪзҡ„еҺҹзҗҶйғҪжҳҜдёҖж ·зҡ„гҖӮ

жҳ е°„еҷЁ

#{}е’Ң${}зҡ„еҢәеҲ«

  • #{}жҳҜеҚ дҪҚз¬ҰпјҢйў„зј–иҜ‘еӨ„зҗҶпјӣ${}жҳҜжӢјжҺҘз¬ҰпјҢеӯ—з¬ҰдёІжӣҝжҚўпјҢжІЎжңүйў„зј–иҜ‘еӨ„зҗҶгҖӮ

  • MybatisеңЁеӨ„зҗҶ#{}ж—¶пјҢ#{}дј е…ҘеҸӮж•°жҳҜд»Ҙеӯ—з¬ҰдёІдј е…ҘпјҢдјҡе°ҶSQLдёӯзҡ„#{}жӣҝжҚўдёә?еҸ·пјҢи°ғз”ЁPreparedStatementзҡ„setж–№жі•жқҘиөӢеҖјгҖӮ

  • MybatisеңЁеӨ„зҗҶж—¶пјҢжҳҜеҺҹеҖјдј е…ҘпјҢе°ұжҳҜжҠҠ{}ж—¶пјҢжҳҜеҺҹеҖјдј е…ҘпјҢе°ұжҳҜжҠҠж—¶пјҢжҳҜеҺҹеҖјдј е…ҘпјҢе°ұжҳҜжҠҠ{}жӣҝжҚўжҲҗеҸҳйҮҸзҡ„еҖјпјҢзӣёеҪ“дәҺJDBCдёӯзҡ„Statementзј–иҜ‘

  • еҸҳйҮҸжӣҝжҚўеҗҺпјҢ#{} еҜ№еә”зҡ„еҸҳйҮҸиҮӘеҠЁеҠ дёҠеҚ•еј•еҸ· вҖҳвҖҷпјӣеҸҳйҮҸжӣҝжҚўеҗҺпјҢ${} еҜ№еә”зҡ„еҸҳйҮҸдёҚдјҡеҠ дёҠеҚ•еј•еҸ· вҖҳвҖҷ

  • #{} еҸҜд»Ҙжңүж•Ҳзҡ„йҳІжӯўSQLжіЁе…ҘпјҢжҸҗй«ҳзі»з»ҹе®үе…ЁжҖ§пјӣ${} дёҚиғҪйҳІжӯўSQL жіЁе…Ҙ

  • #{} зҡ„еҸҳйҮҸжӣҝжҚўжҳҜеңЁDBMS дёӯпјӣ${} зҡ„еҸҳйҮҸжӣҝжҚўжҳҜеңЁ DBMS еӨ–

е®һзҺ°Mybatisзҡ„InterceptorжҺҘеҸЈе№¶еӨҚеҶҷintercept()ж–№жі•пјҢз„¶еҗҺеңЁз»ҷжҸ’д»¶зј–еҶҷжіЁи§ЈпјҢжҢҮе®ҡиҰҒжӢҰжҲӘе“ӘдёҖдёӘжҺҘеҸЈзҡ„е“Әдәӣж–№жі•еҚіеҸҜпјҢи®°дҪҸпјҢеҲ«еҝҳдәҶеңЁй…ҚзҪ®ж–Үд»¶дёӯй…ҚзҪ®дҪ зј–еҶҷзҡ„жҸ’д»¶гҖӮ

зј“еӯҳ

Mybatisзҡ„дёҖзә§гҖҒдәҢзә§зј“еӯҳ

1пјүдёҖзә§зј“еӯҳ: еҹәдәҺ PerpetualCache зҡ„ HashMap жң¬ең°зј“еӯҳпјҢе…¶еӯҳеӮЁдҪңз”Ёеҹҹдёә SessionпјҢеҪ“ Session flush жҲ– close д№ӢеҗҺпјҢиҜҘ Session дёӯзҡ„жүҖжңү Cache е°ұе°Ҷжё…з©әпјҢй»ҳи®Өжү“ејҖдёҖзә§зј“еӯҳгҖӮ

2пјүдәҢзә§зј“еӯҳдёҺдёҖзә§зј“еӯҳе…¶жңәеҲ¶зӣёеҗҢпјҢй»ҳи®Өд№ҹжҳҜйҮҮз”Ё PerpetualCacheпјҢHashMap еӯҳеӮЁпјҢдёҚеҗҢеңЁдәҺе…¶еӯҳеӮЁдҪңз”Ёеҹҹдёә Mapper(Namespace)пјҢе№¶дё”еҸҜиҮӘе®ҡд№үеӯҳеӮЁжәҗпјҢеҰӮ EhcacheгҖӮй»ҳи®ӨдёҚжү“ејҖдәҢзә§зј“еӯҳпјҢиҰҒејҖеҗҜдәҢзә§зј“еӯҳпјҢдҪҝз”ЁдәҢзә§зј“еӯҳеұһжҖ§зұ»йңҖиҰҒе®һзҺ°SerializableеәҸеҲ—еҢ–жҺҘеҸЈ(еҸҜз”ЁжқҘдҝқеӯҳеҜ№иұЎзҡ„зҠ¶жҖҒ),еҸҜеңЁе®ғзҡ„жҳ е°„ж–Үд»¶дёӯй…ҚзҪ®<cache/> пјӣ

3пјүеҜ№дәҺзј“еӯҳж•°жҚ®жӣҙж–°жңәеҲ¶пјҢеҪ“жҹҗдёҖдёӘдҪңз”Ёеҹҹ(дёҖзә§зј“еӯҳ Session/дәҢзә§зј“еӯҳNamespaces)зҡ„иҝӣиЎҢдәҶC/U/D ж“ҚдҪңеҗҺпјҢй»ҳи®ӨиҜҘдҪңз”ЁеҹҹдёӢжүҖжңү select дёӯзҡ„зј“еӯҳе°Ҷиў« clearгҖӮ
