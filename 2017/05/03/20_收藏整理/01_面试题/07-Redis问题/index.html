дҪңиҖ…:Jerry ж—¶й—ҙ:2017-05-03 00:05:15 и®ҝй—®:

жҰӮиҝ°
д»Җд№ҲжҳҜRedis
Redis(Remote Dictionary Server) жҳҜдёҖдёӘдҪҝз”Ё C иҜӯиЁҖзј–еҶҷзҡ„пјҢејҖжәҗзҡ„пјҲBSDи®ёеҸҜпјүй«ҳжҖ§иғҪйқһе…ізі»еһӢпјҲNoSQLпјүзҡ„й”®еҖјеҜ№ж•°жҚ®еә“гҖӮ

Redis еҸҜд»ҘеӯҳеӮЁй”®е’Ңдә”з§ҚдёҚеҗҢзұ»еһӢзҡ„еҖјд№Ӣй—ҙзҡ„жҳ е°„гҖӮй”®зҡ„зұ»еһӢеҸӘиғҪдёәеӯ—з¬ҰдёІпјҢеҖјж”ҜжҢҒдә”з§Қж•°жҚ®зұ»еһӢпјҡеӯ—з¬ҰдёІгҖҒеҲ—иЎЁгҖҒйӣҶеҗҲгҖҒж•ЈеҲ—иЎЁгҖҒжңүеәҸйӣҶеҗҲгҖӮ

дёҺдј з»ҹж•°жҚ®еә“дёҚеҗҢзҡ„жҳҜ Redis зҡ„ж•°жҚ®жҳҜеӯҳеңЁеҶ…еӯҳдёӯзҡ„пјҢжүҖд»ҘиҜ»еҶҷйҖҹеәҰйқһеёёеҝ«пјҢеӣ жӯӨ redis иў«е№ҝжіӣеә”з”ЁдәҺзј“еӯҳж–№еҗ‘пјҢжҜҸз§’еҸҜд»ҘеӨ„зҗҶи¶…иҝҮ 10дёҮж¬ЎиҜ»еҶҷж“ҚдҪңпјҢжҳҜе·ІзҹҘжҖ§иғҪжңҖеҝ«зҡ„Key-Value DBгҖӮеҸҰеӨ–пјҢRedis д№ҹз»Ҹеёёз”ЁжқҘеҒҡеҲҶеёғејҸй”ҒгҖӮйҷӨжӯӨд№ӢеӨ–пјҢRedis ж”ҜжҢҒдәӢеҠЎ гҖҒжҢҒд№…еҢ–гҖҒLUAи„ҡжң¬гҖҒLRUй©ұеҠЁдәӢд»¶гҖҒеӨҡз§ҚйӣҶзҫӨж–№жЎҲгҖӮ

Redisжңүе“ӘдәӣдјҳзјәзӮ№
дјҳзӮ№

иҜ»еҶҷжҖ§иғҪдјҳејӮпјҢ RedisиғҪиҜ»зҡ„йҖҹеәҰжҳҜ110000ж¬Ў/sпјҢеҶҷзҡ„йҖҹеәҰжҳҜ81000ж¬Ў/sгҖӮ
ж”ҜжҢҒж•°жҚ®жҢҒд№…еҢ–пјҢж”ҜжҢҒAOFе’ҢRDBдёӨз§ҚжҢҒд№…еҢ–ж–№ејҸгҖӮ
ж”ҜжҢҒдәӢеҠЎпјҢRedisзҡ„жүҖжңүж“ҚдҪңйғҪжҳҜеҺҹеӯҗжҖ§зҡ„пјҢеҗҢж—¶Redisиҝҳж”ҜжҢҒеҜ№еҮ дёӘж“ҚдҪңеҗҲе№¶еҗҺзҡ„еҺҹеӯҗжҖ§жү§иЎҢгҖӮ
ж•°жҚ®з»“жһ„дё°еҜҢпјҢйҷӨдәҶж”ҜжҢҒstringзұ»еһӢзҡ„valueеӨ–иҝҳж”ҜжҢҒhashгҖҒsetгҖҒzsetгҖҒlistзӯүж•°жҚ®з»“жһ„гҖӮ
ж”ҜжҢҒдё»д»ҺеӨҚеҲ¶пјҢдё»жңәдјҡиҮӘеҠЁе°Ҷж•°жҚ®еҗҢжӯҘеҲ°д»ҺжңәпјҢеҸҜд»ҘиҝӣиЎҢиҜ»еҶҷеҲҶзҰ»гҖӮ
зјәзӮ№

ж•°жҚ®еә“е®№йҮҸеҸ—еҲ°зү©зҗҶеҶ…еӯҳзҡ„йҷҗеҲ¶пјҢдёҚиғҪз”ЁдҪңжө·йҮҸж•°жҚ®зҡ„й«ҳжҖ§иғҪиҜ»еҶҷпјҢеӣ жӯӨRedisйҖӮеҗҲзҡ„еңәжҷҜдё»иҰҒеұҖйҷҗеңЁиҫғе°Ҹж•°жҚ®йҮҸзҡ„й«ҳжҖ§иғҪж“ҚдҪңе’Ңиҝҗз®—дёҠгҖӮ
Redis дёҚе…·еӨҮиҮӘеҠЁе®№й”ҷе’ҢжҒўеӨҚеҠҹиғҪпјҢдё»жңәд»Һжңәзҡ„е®•жңәйғҪдјҡеҜјиҮҙеүҚз«ҜйғЁеҲҶиҜ»еҶҷиҜ·жұӮеӨұиҙҘпјҢйңҖиҰҒзӯүеҫ…жңәеҷЁйҮҚеҗҜжҲ–иҖ…жүӢеҠЁеҲҮжҚўеүҚз«Ҝзҡ„IPжүҚиғҪжҒўеӨҚгҖӮ
дё»жңәе®•жңәпјҢе®•жңәеүҚжңүйғЁеҲҶж•°жҚ®жңӘиғҪеҸҠж—¶еҗҢжӯҘеҲ°д»ҺжңәпјҢеҲҮжҚўIPеҗҺиҝҳдјҡеј•е…Ҙж•°жҚ®дёҚдёҖиҮҙзҡ„й—®йўҳпјҢйҷҚдҪҺдәҶзі»з»ҹзҡ„еҸҜз”ЁжҖ§гҖӮ
Redis иҫғйҡҫж”ҜжҢҒеңЁзәҝжү©е®№пјҢеңЁйӣҶзҫӨе®№йҮҸиҫҫеҲ°дёҠйҷҗж—¶еңЁзәҝжү©е®№дјҡеҸҳеҫ—еҫҲеӨҚжқӮгҖӮдёәйҒҝе…ҚиҝҷдёҖй—®йўҳпјҢиҝҗз»ҙдәәе‘ҳеңЁзі»з»ҹдёҠзәҝж—¶еҝ…йЎ»зЎ®дҝқжңүи¶іеӨҹзҡ„з©әй—ҙпјҢиҝҷеҜ№иө„жәҗйҖ жҲҗдәҶеҫҲеӨ§зҡ„жөӘиҙ№гҖӮ
дёәд»Җд№ҲиҰҒз”Ё Redis /дёәд»Җд№ҲиҰҒз”Ёзј“еӯҳ
дё»иҰҒд»ҺвҖңй«ҳжҖ§иғҪвҖқе’ҢвҖңй«ҳе№¶еҸ‘вҖқиҝҷдёӨзӮ№жқҘзңӢеҫ…иҝҷдёӘй—®йўҳгҖӮ

й«ҳжҖ§иғҪпјҡ

еҒҮеҰӮз”ЁжҲ·з¬¬дёҖж¬Ўи®ҝй—®ж•°жҚ®еә“дёӯзҡ„жҹҗдәӣж•°жҚ®гҖӮиҝҷдёӘиҝҮзЁӢдјҡжҜ”иҫғж…ўпјҢеӣ дёәжҳҜд»ҺзЎ¬зӣҳдёҠиҜ»еҸ–зҡ„гҖӮе°ҶиҜҘз”ЁжҲ·и®ҝй—®зҡ„ж•°жҚ®еӯҳеңЁж•°зј“еӯҳдёӯпјҢиҝҷж ·дёӢдёҖж¬ЎеҶҚи®ҝй—®иҝҷдәӣж•°жҚ®зҡ„ж—¶еҖҷе°ұеҸҜд»ҘзӣҙжҺҘд»Һзј“еӯҳдёӯиҺ·еҸ–дәҶгҖӮж“ҚдҪңзј“еӯҳе°ұжҳҜзӣҙжҺҘж“ҚдҪңеҶ…еӯҳпјҢжүҖд»ҘйҖҹеәҰзӣёеҪ“еҝ«гҖӮеҰӮжһңж•°жҚ®еә“дёӯзҡ„еҜ№еә”ж•°жҚ®ж”№еҸҳзҡ„д№ӢеҗҺпјҢеҗҢжӯҘж”№еҸҳзј“еӯҳдёӯзӣёеә”зҡ„ж•°жҚ®еҚіеҸҜпјҒ

й«ҳе№¶еҸ‘пјҡ

зӣҙжҺҘж“ҚдҪңзј“еӯҳиғҪеӨҹжүҝеҸ—зҡ„иҜ·жұӮжҳҜиҝңиҝңеӨ§дәҺзӣҙжҺҘи®ҝй—®ж•°жҚ®еә“зҡ„пјҢжүҖд»ҘжҲ‘д»¬еҸҜд»ҘиҖғиҷ‘жҠҠж•°жҚ®еә“дёӯзҡ„йғЁеҲҶж•°жҚ®иҪ¬з§»еҲ°зј“еӯҳдёӯеҺ»пјҢиҝҷж ·з”ЁжҲ·зҡ„дёҖйғЁеҲҶиҜ·жұӮдјҡзӣҙжҺҘеҲ°зј“еӯҳиҝҷйҮҢиҖҢдёҚз”Ёз»ҸиҝҮж•°жҚ®еә“гҖӮ

дёәд»Җд№ҲиҰҒз”Ё Redis иҖҢдёҚз”Ё map/guava еҒҡзј“еӯҳ?
зј“еӯҳеҲҶдёәжң¬ең°зј“еӯҳе’ҢеҲҶеёғејҸзј“еӯҳгҖӮд»Ҙ Java дёәдҫӢпјҢдҪҝз”ЁиҮӘеёҰзҡ„ map жҲ–иҖ… guava е®һзҺ°зҡ„жҳҜжң¬ең°зј“еӯҳпјҢжңҖдё»иҰҒзҡ„зү№зӮ№жҳҜиҪ»йҮҸд»ҘеҸҠеҝ«йҖҹпјҢз”ҹе‘Ҫе‘ЁжңҹйҡҸзқҖ jvm зҡ„й”ҖжҜҒиҖҢз»“жқҹпјҢе№¶дё”еңЁеӨҡе®һдҫӢзҡ„жғ…еҶөдёӢпјҢжҜҸдёӘе®һдҫӢйғҪйңҖиҰҒеҗ„иҮӘдҝқеӯҳдёҖд»Ҫзј“еӯҳпјҢзј“еӯҳдёҚе…·жңүдёҖиҮҙжҖ§гҖӮ

дҪҝз”ЁkeysжҢҮд»ӨеҸҜд»Ҙжү«еҮәжҢҮе®ҡжЁЎејҸзҡ„keyеҲ—иЎЁгҖӮ

иҝҷдёӘж—¶еҖҷдҪ иҰҒеӣһзӯ”redisе…ій”®зҡ„дёҖдёӘзү№жҖ§пјҡredisзҡ„еҚ•зәҝзЁӢзҡ„гҖӮkeysжҢҮд»ӨдјҡеҜјиҮҙзәҝзЁӢйҳ»еЎһдёҖж®өж—¶й—ҙпјҢзәҝдёҠжңҚеҠЎдјҡеҒңйЎҝпјҢзӣҙеҲ°жҢҮд»Өжү§иЎҢе®ҢжҜ•пјҢжңҚеҠЎжүҚиғҪжҒўеӨҚгҖӮиҝҷдёӘж—¶еҖҷеҸҜд»ҘдҪҝз”ЁscanжҢҮд»ӨпјҢscanжҢҮд»ӨеҸҜд»Ҙж— йҳ»еЎһзҡ„жҸҗеҸ–еҮәжҢҮе®ҡжЁЎејҸзҡ„keyеҲ—иЎЁпјҢдҪҶжҳҜдјҡжңүдёҖе®ҡзҡ„йҮҚеӨҚжҰӮзҺҮпјҢеңЁе®ўжҲ·з«ҜеҒҡдёҖж¬ЎеҺ»йҮҚе°ұеҸҜд»ҘдәҶпјҢдҪҶжҳҜж•ҙдҪ“жүҖиҠұиҙ№зҡ„ж—¶й—ҙдјҡжҜ”зӣҙжҺҘз”ЁkeysжҢҮд»Өй•ҝгҖӮ
