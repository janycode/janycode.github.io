2. дҪҝз”ЁSpringжЎҶжһ¶зҡ„еҘҪеӨ„жҳҜд»Җд№Ҳпјҹ

иҪ»йҮҸпјҡSpring жҳҜиҪ»йҮҸзҡ„пјҢеҹәжң¬зҡ„зүҲжң¬еӨ§зәҰ2MBгҖӮ
жҺ§еҲ¶еҸҚиҪ¬пјҡSpringйҖҡиҝҮжҺ§еҲ¶еҸҚиҪ¬е®һзҺ°дәҶжқҫж•ЈиҖҰеҗҲпјҢеҜ№иұЎд»¬з»ҷеҮәе®ғд»¬зҡ„дҫқиө–пјҢиҖҢдёҚжҳҜеҲӣе»әжҲ–жҹҘжүҫдҫқиө–зҡ„еҜ№иұЎд»¬гҖӮ
йқўеҗ‘еҲҮйқўзҡ„зј–зЁӢ(AOP)пјҡSpringж”ҜжҢҒйқўеҗ‘еҲҮйқўзҡ„зј–зЁӢпјҢе№¶дё”жҠҠеә”з”ЁдёҡеҠЎйҖ»иҫ‘е’Ңзі»з»ҹжңҚеҠЎеҲҶејҖгҖӮ
е®№еҷЁпјҡSpring еҢ…еҗ«е№¶з®ЎзҗҶеә”з”ЁдёӯеҜ№иұЎзҡ„з”ҹе‘Ҫе‘Ёжңҹе’Ңй…ҚзҪ®гҖӮ

дәӢеҠЎз®ЎзҗҶпјҡSpring жҸҗдҫӣдёҖдёӘжҢҒз»ӯзҡ„дәӢеҠЎз®ЎзҗҶжҺҘеҸЈпјҢеҸҜд»Ҙжү©еұ•еҲ°дёҠиҮіжң¬ең°дәӢеҠЎдёӢиҮіе…ЁеұҖдәӢеҠЎпјҲJTAпјүгҖӮ

4. ж ёеҝғе®№еҷЁпјҲеә”з”ЁдёҠдёӢж–Ү) жЁЎеқ—

5. BeanFactory вҖ“ BeanFactory е®һзҺ°дёҫдҫӢ

Bean е·ҘеҺӮжҳҜе·ҘеҺӮжЁЎејҸзҡ„дёҖдёӘе®һзҺ°пјҢжҸҗдҫӣдәҶжҺ§еҲ¶еҸҚиҪ¬еҠҹиғҪпјҢз”ЁжқҘжҠҠеә”з”Ёзҡ„й…ҚзҪ®е’Ңдҫқиө–д»ҺжӯЈзңҹзҡ„еә”з”Ёд»Јз ҒдёӯеҲҶзҰ»гҖӮ

жңҖеёёз”Ёзҡ„ BeanFactory е®һзҺ°жҳҜ XmlBeanFactory зұ»гҖӮ

6. XMLBeanFactory

жңҖеёёз”Ёзҡ„е°ұжҳҜ org.springframework.beans.factory.xml.XmlBeanFactory пјҢе®ғж №жҚ®XMLж–Үд»¶дёӯзҡ„е®ҡд№үеҠ иҪҪbeansгҖӮиҜҘе®№еҷЁд»Һ XML ж–Үд»¶иҜ»еҸ–й…ҚзҪ®е…ғж•°жҚ®е№¶з”Ёе®ғеҺ»еҲӣе»әдёҖдёӘе®Ңе…Ёй…ҚзҪ®зҡ„зі»з»ҹжҲ–еә”з”ЁгҖӮ

7. и§ЈйҮҠAOPжЁЎеқ—

AOPжЁЎеқ—з”ЁдәҺеҸ‘з»ҷжҲ‘д»¬зҡ„Springеә”з”ЁеҒҡйқўеҗ‘еҲҮйқўзҡ„ејҖеҸ‘пјҢ еҫҲеӨҡж”ҜжҢҒз”ұAOPиҒ”зӣҹжҸҗдҫӣпјҢиҝҷж ·е°ұзЎ®дҝқдәҶSpringе’Ңе…¶д»–AOPжЎҶжһ¶зҡ„е…ұйҖҡжҖ§гҖӮиҝҷдёӘжЁЎеқ—е°Ҷе…ғж•°жҚ®зј–зЁӢеј•е…ҘSpringгҖӮ

8. и§ЈйҮҠJDBCжҠҪиұЎе’ҢDAOжЁЎеқ—

йҖҡиҝҮдҪҝз”ЁJDBCжҠҪиұЎе’ҢDAOжЁЎеқ—пјҢдҝқиҜҒж•°жҚ®еә“д»Јз Ғзҡ„з®ҖжҙҒпјҢе№¶иғҪйҒҝе…Қж•°жҚ®еә“иө„жәҗй”ҷиҜҜе…ій—ӯеҜјиҮҙзҡ„й—®йўҳпјҢе®ғеңЁеҗ„з§ҚдёҚеҗҢзҡ„ж•°жҚ®еә“зҡ„й”ҷиҜҜдҝЎжҒҜд№ӢдёҠпјҢжҸҗдҫӣдәҶдёҖдёӘз»ҹдёҖзҡ„ејӮеёёи®ҝй—®еұӮгҖӮе®ғиҝҳеҲ©з”ЁSpringзҡ„AOP жЁЎеқ—з»ҷSpringеә”з”Ёдёӯзҡ„еҜ№иұЎжҸҗдҫӣдәӢеҠЎз®ЎзҗҶжңҚеҠЎгҖӮ

9. и§ЈйҮҠеҜ№иұЎ/е…ізі»жҳ е°„йӣҶжҲҗжЁЎеқ—

Spring йҖҡиҝҮжҸҗдҫӣORMжЁЎеқ—пјҢж”ҜжҢҒжҲ‘д»¬еңЁзӣҙжҺҘJDBCд№ӢдёҠдҪҝз”ЁдёҖдёӘеҜ№иұЎ/е…ізі»жҳ е°„жҳ е°„(ORM)е·Ҙе…·пјҢSpring ж”ҜжҢҒйӣҶжҲҗдё»жөҒзҡ„ORMжЎҶжһ¶пјҢеҰӮHiberate,JDOе’Ң iBATIS SQL MapsгҖӮSpringзҡ„дәӢеҠЎз®ЎзҗҶеҗҢж ·ж”ҜжҢҒд»ҘдёҠжүҖжңүORMжЎҶжһ¶еҸҠJDBCгҖӮ

20. д»Җд№ҲжҳҜSpring beans?

Spring beans жҳҜйӮЈдәӣеҪўжҲҗSpringеә”з”Ёзҡ„дё»е№Ізҡ„javaеҜ№иұЎгҖӮе®ғд»¬иў«Spring IOCе®№еҷЁеҲқе§ӢеҢ–пјҢиЈ…й…ҚпјҢе’Ңз®ЎзҗҶгҖӮиҝҷдәӣbeansйҖҡиҝҮе®№еҷЁдёӯй…ҚзҪ®зҡ„е…ғж•°жҚ®еҲӣе»әгҖӮжҜ”еҰӮпјҢд»ҘXMLж–Үд»¶дёӯ зҡ„еҪўејҸе®ҡд№үгҖӮ

23. дҪ жҖҺж ·е®ҡд№үзұ»зҡ„дҪңз”Ёеҹҹ?

еҪ“е®ҡд№үдёҖдёӘ еңЁSpringйҮҢпјҢжҲ‘д»¬иҝҳиғҪз»ҷиҝҷдёӘbeanеЈ°жҳҺдёҖдёӘдҪңз”ЁеҹҹгҖӮе®ғеҸҜд»ҘйҖҡиҝҮbean е®ҡд№үдёӯзҡ„scopeеұһжҖ§жқҘе®ҡд№үгҖӮеҰӮпјҢеҪ“SpringиҰҒеңЁйңҖиҰҒзҡ„ж—¶еҖҷжҜҸж¬Ўз”ҹдә§дёҖдёӘж–°зҡ„beanе®һдҫӢпјҢbeanзҡ„scopeеұһжҖ§иў«жҢҮе®ҡдёәprototypeгҖӮеҸҰдёҖж–№йқўпјҢдёҖдёӘbeanжҜҸж¬ЎдҪҝз”Ёзҡ„ж—¶еҖҷеҝ…йЎ»иҝ”еӣһеҗҢдёҖдёӘе®һдҫӢпјҢиҝҷдёӘbeanзҡ„scope еұһжҖ§ еҝ…йЎ»и®ҫдёә singletonгҖӮ

24. и§ЈйҮҠSpringж”ҜжҢҒзҡ„еҮ з§Қbeanзҡ„дҪңз”ЁеҹҹгҖӮ

SpringжЎҶжһ¶ж”ҜжҢҒд»ҘдёӢдә”з§Қbeanзҡ„дҪңз”Ёеҹҹпјҡ

singleton : beanеңЁжҜҸдёӘSpring ioc е®№еҷЁдёӯеҸӘжңүдёҖдёӘе®һдҫӢгҖӮ
prototypeпјҡдёҖдёӘbeanзҡ„е®ҡд№үеҸҜд»ҘжңүеӨҡдёӘе®һдҫӢгҖӮ
requestпјҡжҜҸж¬ЎhttpиҜ·жұӮйғҪдјҡеҲӣе»әдёҖдёӘbeanпјҢиҜҘдҪңз”Ёеҹҹд»…еңЁеҹәдәҺwebзҡ„Spring ApplicationContextжғ…еҪўдёӢжңүж•ҲгҖӮ

26. и§ЈйҮҠSpringжЎҶжһ¶дёӯbeanзҡ„з”ҹе‘Ҫе‘ЁжңҹгҖӮ

03-SpringжіЁи§Ј

35. д»Җд№ҲжҳҜеҹәдәҺJavaзҡ„SpringжіЁи§Јй…ҚзҪ®? з»ҷдёҖдәӣжіЁи§Јзҡ„дҫӢеӯҗ.

д»Ҙ@Configuration жіЁи§ЈдёәдҫӢпјҢе®ғз”ЁжқҘж Үи®°зұ»еҸҜд»ҘеҪ“еҒҡдёҖдёӘbeanзҡ„е®ҡд№үпјҢиў«Spring IOCе®№еҷЁдҪҝз”ЁгҖӮеҸҰдёҖдёӘдҫӢеӯҗжҳҜ@BeanжіЁи§ЈпјҢе®ғиЎЁзӨәжӯӨж–№жі•е°ҶиҰҒиҝ”еӣһдёҖдёӘеҜ№иұЎпјҢдҪңдёәдёҖдёӘbeanжіЁеҶҢиҝӣSpringеә”з”ЁдёҠдёӢж–ҮгҖӮ

36. д»Җд№ҲжҳҜеҹәдәҺжіЁи§Јзҡ„е®№еҷЁй…ҚзҪ®?

зӣёеҜ№дәҺXMLж–Үд»¶пјҢжіЁи§ЈеһӢзҡ„й…ҚзҪ®дҫқиө–дәҺйҖҡиҝҮеӯ—иҠӮз Ғе…ғж•°жҚ®иЈ…й…Қз»„д»¶пјҢиҖҢйқһе°–жӢ¬еҸ·зҡ„еЈ°жҳҺгҖӮ

ејҖеҸ‘иҖ…йҖҡиҝҮеңЁзӣёеә”зҡ„зұ»пјҢж–№жі•жҲ–еұһжҖ§дёҠдҪҝз”ЁжіЁи§Јзҡ„ж–№ејҸпјҢзӣҙжҺҘз»„д»¶зұ»дёӯиҝӣиЎҢй…ҚзҪ®пјҢиҖҢдёҚжҳҜдҪҝз”ЁxmlиЎЁиҝ°beanзҡ„иЈ…й…Қе…ізі»гҖӮ

37. жҖҺж ·ејҖеҗҜжіЁи§ЈиЈ…й…Қпјҹ

жіЁи§ЈиЈ…й…ҚеңЁй»ҳи®Өжғ…еҶөдёӢжҳҜдёҚејҖеҗҜзҡ„пјҢдёәдәҶдҪҝз”ЁжіЁи§ЈиЈ…й…ҚпјҢжҲ‘д»¬еҝ…йЎ»еңЁSpringй…ҚзҪ®ж–Үд»¶дёӯй…ҚзҪ® context:annotation-config/е…ғзҙ гҖӮ

38. @Required жіЁи§Ј

39. @Autowired жіЁи§Ј

@Autowired жіЁи§ЈжҸҗдҫӣдәҶжӣҙз»ҶзІ’еәҰзҡ„жҺ§еҲ¶пјҢеҢ…жӢ¬еңЁдҪ•еӨ„д»ҘеҸҠеҰӮдҪ•е®ҢжҲҗиҮӘеҠЁиЈ…й…ҚгҖӮе®ғзҡ„з”Ёжі•е’Ң@RequiredдёҖж ·пјҢдҝ®йҘ°setterж–№жі•гҖҒжһ„йҖ еҷЁгҖҒеұһжҖ§жҲ–иҖ…е…·жңүд»»ж„ҸеҗҚз§°е’Ң/жҲ–еӨҡдёӘеҸӮж•°зҡ„PNж–№жі•гҖӮ

40. @Qualifier жіЁи§Ј

еҪ“жңүеӨҡдёӘзӣёеҗҢзұ»еһӢзҡ„beanеҚҙеҸӘжңүдёҖдёӘйңҖиҰҒиҮӘеҠЁиЈ…й…Қж—¶пјҢе°Ҷ@Qualifier жіЁи§Је’Ң@Autowire жіЁи§Јз»“еҗҲдҪҝз”Ёд»Ҙж¶ҲйҷӨиҝҷз§Қж··ж·ҶпјҢжҢҮе®ҡйңҖиҰҒиЈ…й…Қзҡ„зЎ®еҲҮзҡ„beanгҖӮ

41. еңЁSpringжЎҶжһ¶дёӯеҰӮдҪ•жӣҙжңүж•Ҳең°дҪҝз”ЁJDBC?

дҪҝз”ЁSpringJDBC жЎҶжһ¶пјҢиө„жәҗз®ЎзҗҶе’Ңй”ҷиҜҜеӨ„зҗҶзҡ„д»Јд»·йғҪдјҡиў«еҮҸиҪ»гҖӮжүҖд»ҘејҖеҸ‘иҖ…еҸӘйңҖеҶҷstatements е’Ң queriesд»Һж•°жҚ®еӯҳеҸ–ж•°жҚ®пјҢJDBCд№ҹеҸҜд»ҘеңЁSpringжЎҶжһ¶жҸҗдҫӣзҡ„жЁЎжқҝзұ»зҡ„её®еҠ©дёӢжӣҙжңүж•Ҳең°иў«дҪҝз”ЁпјҢиҝҷдёӘжЁЎжқҝеҸ«JdbcTemplate пјҲдҫӢеӯҗи§ҒиҝҷйҮҢhereпјү

43. SpringеҜ№DAOзҡ„ж”ҜжҢҒ

47. Springж”ҜжҢҒзҡ„дәӢеҠЎз®ЎзҗҶзұ»еһӢ

Springж”ҜжҢҒдёӨз§Қзұ»еһӢзҡ„дәӢеҠЎз®ЎзҗҶпјҡ

зј–зЁӢејҸдәӢеҠЎз®ЎзҗҶпјҡиҝҷж„Ҹе‘ідҪ йҖҡиҝҮзј–зЁӢзҡ„ж–№ејҸз®ЎзҗҶдәӢеҠЎпјҢз»ҷдҪ еёҰжқҘжһҒеӨ§зҡ„зҒөжҙ»жҖ§пјҢдҪҶжҳҜйҡҫз»ҙжҠӨгҖӮ
еЈ°жҳҺејҸдәӢеҠЎз®ЎзҗҶпјҡиҝҷж„Ҹе‘ізқҖдҪ еҸҜд»Ҙе°ҶдёҡеҠЎд»Јз Ғе’ҢдәӢеҠЎз®ЎзҗҶеҲҶзҰ»пјҢдҪ еҸӘйңҖз”ЁжіЁи§Је’ҢXMLй…ҚзҪ®жқҘз®ЎзҗҶдәӢеҠЎгҖӮ

48. SpringжЎҶжһ¶зҡ„дәӢеҠЎз®ЎзҗҶжңүе“ӘдәӣдјҳзӮ№пјҹ

е®ғдёәдёҚеҗҢзҡ„дәӢеҠЎAPI еҰӮ JTAпјҢJDBCпјҢHibernateпјҢJPA е’ҢJDOпјҢжҸҗдҫӣдёҖдёӘдёҚеҸҳзҡ„зј–зЁӢжЁЎејҸгҖӮ
е®ғдёәзј–зЁӢејҸдәӢеҠЎз®ЎзҗҶжҸҗдҫӣдәҶдёҖеҘ—з®ҖеҚ•зҡ„APIиҖҢдёҚжҳҜдёҖдәӣеӨҚжқӮзҡ„дәӢеҠЎAPIеҰӮ
е®ғж”ҜжҢҒеЈ°жҳҺејҸдәӢеҠЎз®ЎзҗҶгҖӮ
е®ғе’ҢSpringеҗ„з§Қж•°жҚ®и®ҝй—®жҠҪиұЎеұӮеҫҲеҘҪеҫ—йӣҶжҲҗгҖӮ

49. дҪ жӣҙеҖҫеҗ‘з”ЁйӮЈз§ҚдәӢеҠЎз®ЎзҗҶзұ»еһӢпјҹ

еӨ§еӨҡж•°SpringжЎҶжһ¶зҡ„з”ЁжҲ·йҖүжӢ©еЈ°жҳҺејҸдәӢеҠЎз®ЎзҗҶпјҢеӣ дёәе®ғеҜ№еә”з”Ёд»Јз Ғзҡ„еҪұе“ҚжңҖе°ҸпјҢеӣ жӯӨжӣҙз¬ҰеҗҲдёҖдёӘж— дҫөе…Ҙзҡ„иҪ»йҮҸзә§е®№еҷЁзҡ„жҖқжғігҖӮеЈ°жҳҺејҸдәӢеҠЎз®ЎзҗҶиҰҒдјҳдәҺзј–зЁӢејҸдәӢеҠЎз®ЎзҗҶпјҢиҷҪз„¶жҜ”зј–зЁӢејҸдәӢеҠЎз®ЎзҗҶпјҲиҝҷз§Қж–№ејҸе…Ғи®ёдҪ йҖҡиҝҮд»Јз ҒжҺ§еҲ¶дәӢеҠЎпјүе°‘дәҶдёҖзӮ№зҒөжҙ»жҖ§гҖӮ

50. и§ЈйҮҠAOP

йқўеҗ‘еҲҮйқўзҡ„зј–зЁӢпјҢжҲ–AOPпјҢ жҳҜдёҖз§Қзј–зЁӢжҠҖжңҜпјҢе…Ғи®ёзЁӢеәҸжЁЎеқ—еҢ–жЁӘеҗ‘еҲҮеүІе…іжіЁзӮ№пјҢжҲ–жЁӘеҲҮе…ёеһӢзҡ„иҙЈд»»еҲ’еҲҶпјҢеҰӮж—Ҙеҝ—е’ҢдәӢеҠЎз®ЎзҗҶгҖӮ

51. Aspect еҲҮйқў

AOPж ёеҝғе°ұжҳҜеҲҮйқўпјҢе®ғе°ҶеӨҡдёӘзұ»зҡ„йҖҡз”ЁиЎҢдёәе°ҒиЈ…жҲҗеҸҜйҮҚз”Ёзҡ„жЁЎеқ—пјҢиҜҘжЁЎеқ—еҗ«жңүдёҖз»„APIжҸҗдҫӣжЁӘеҲҮеҠҹиғҪгҖӮжҜ”еҰӮпјҢдёҖдёӘж—Ҙеҝ—жЁЎеқ—еҸҜд»Ҙиў«з§°дҪңж—Ҙеҝ—зҡ„AOPеҲҮйқўгҖӮж №жҚ®йңҖжұӮзҡ„дёҚеҗҢпјҢдёҖдёӘеә”з”ЁзЁӢеәҸеҸҜд»ҘжңүиӢҘе№ІеҲҮйқўгҖӮеңЁSpring AOPдёӯпјҢеҲҮйқўйҖҡиҝҮеёҰжңү@AspectжіЁи§Јзҡ„зұ»е®һзҺ°гҖӮ

52. еңЁSpring AOP дёӯпјҢе…іжіЁзӮ№е’ҢжЁӘеҲҮе…іжіЁзҡ„еҢәеҲ«жҳҜд»Җд№Ҳпјҹ

е…іжіЁзӮ№жҳҜеә”з”ЁдёӯдёҖдёӘжЁЎеқ—зҡ„иЎҢдёәпјҢдёҖдёӘе…іжіЁзӮ№еҸҜиғҪдјҡиў«е®ҡд№үжҲҗдёҖдёӘжҲ‘д»¬жғіе®һзҺ°зҡ„дёҖдёӘеҠҹиғҪгҖӮ
жЁӘеҲҮе…іжіЁзӮ№жҳҜдёҖдёӘе…іжіЁзӮ№пјҢжӯӨе…іжіЁзӮ№жҳҜж•ҙдёӘеә”з”ЁйғҪдјҡдҪҝз”Ёзҡ„еҠҹиғҪпјҢе№¶еҪұе“Қж•ҙдёӘеә”з”ЁпјҢжҜ”еҰӮж—Ҙеҝ—пјҢе®үе…Ёе’Ңж•°жҚ®дј иҫ“пјҢеҮ д№Һеә”з”Ёзҡ„жҜҸдёӘжЁЎеқ—йғҪйңҖиҰҒзҡ„еҠҹиғҪгҖӮеӣ жӯӨиҝҷдәӣйғҪеұһдәҺжЁӘеҲҮе…іжіЁзӮ№гҖӮ

53. иҝһжҺҘзӮ№

иҝһжҺҘзӮ№д»ЈиЎЁдёҖдёӘеә”з”ЁзЁӢеәҸзҡ„жҹҗдёӘдҪҚзҪ®пјҢеңЁиҝҷдёӘдҪҚзҪ®жҲ‘д»¬еҸҜд»ҘжҸ’е…ҘдёҖдёӘAOPеҲҮйқўпјҢе®ғе®һйҷ…дёҠжҳҜдёӘеә”з”ЁзЁӢеәҸжү§иЎҢSpring AOPзҡ„дҪҚзҪ®гҖӮ

54. йҖҡзҹҘ

йҖҡзҹҘжҳҜдёӘеңЁж–№жі•жү§иЎҢеүҚжҲ–жү§иЎҢеҗҺиҰҒеҒҡзҡ„еҠЁдҪңпјҢе®һйҷ…дёҠжҳҜзЁӢеәҸжү§иЎҢж—¶иҰҒйҖҡиҝҮSpringAOPжЎҶжһ¶и§ҰеҸ‘зҡ„д»Јз Ғж®өгҖӮ

SpringеҲҮйқўеҸҜд»Ҙеә”з”Ёдә”з§Қзұ»еһӢзҡ„йҖҡзҹҘпјҡ

beforeпјҡеүҚзҪ®йҖҡзҹҘпјҢеңЁдёҖдёӘж–№жі•жү§иЎҢеүҚиў«и°ғз”ЁгҖӮ
after: еңЁж–№жі•жү§иЎҢд№ӢеҗҺи°ғз”Ёзҡ„йҖҡзҹҘпјҢж— и®әж–№жі•жү§иЎҢжҳҜеҗҰжҲҗеҠҹгҖӮ
after-returning: д»…еҪ“ж–№жі•жҲҗеҠҹе®ҢжҲҗеҗҺжү§иЎҢзҡ„йҖҡзҹҘгҖӮ
after-throwing: еңЁж–№жі•жҠӣеҮәејӮеёёйҖҖеҮәж—¶жү§иЎҢзҡ„йҖҡзҹҘгҖӮ
around: еңЁж–№жі•жү§иЎҢд№ӢеүҚе’Ңд№ӢеҗҺи°ғз”Ёзҡ„йҖҡзҹҘгҖӮ
