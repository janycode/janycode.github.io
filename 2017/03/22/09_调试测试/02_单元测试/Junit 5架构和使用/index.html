1. JUnit 5 жһ¶жһ„

Junit5 = JUnit Platform + JUnit Jupiter + JUnit Vintage

дҪҝз”Ё JUnit Jupiter зј–еҶҷжөӢиҜ•еҶ…е®№

JUnit Jupiter еҢ…еҗ«дёӨдёӘз»„д»¶пјҡAPI е’Ң Test EngineгҖӮ

  • дҪҝз”Ё API пјҲ йҖҡиҝҮжіЁи§ЈгҖҒж–ӯиЁҖгҖҒеӣһи°ғзӯүпјү еҲӣе»әеҚ•е…ғжөӢиҜ•гҖӮ
  • дҪҝз”Ё Test Engine еҸ‘зҺ°е’Ңжү§иЎҢ JUnit Jupiter еҚ•е…ғжөӢиҜ•гҖӮ
дҪҝз”Ё JUnit Platform иҝҗиЎҢжөӢиҜ•

JUnit Platform еҢ…еҗ« APIгҖҒTest Engine е’Ң LauncherгҖӮ

иҝҗиЎҢеҚ•е…ғжөӢиҜ•зҡ„иҝҮзЁӢеҲҶдёәдёӨйғЁеҲҶпјҡ

  • еҸ‘зҺ°жөӢиҜ•е’ҢеҲӣе»әжөӢиҜ•и®ЎеҲ’
    • жҸҗдҫӣз”ұдёҖдёӘ TestEngine е®һзҺ°зҡ„пјҢз”ЁдәҺеҸ‘зҺ°жөӢиҜ•е’ҢеҲӣе»әжөӢиҜ•и®ЎеҲ’зҡ„ API
    • дҪҝз”Ё IDE е’Ңжһ„е»әе·Ҙе…·еҸ‘иө·жөӢиҜ•еҸ‘зҺ°жөҒзЁӢ
    • ж №жҚ®жөӢиҜ•и§„иҢғеҲӣе»әжөӢиҜ•и®ЎеҲ’
  • еҗҜеҠЁжөӢиҜ•и®ЎеҲ’пјҢд»Ҙжү§иЎҢжөӢиҜ•е’Ңеҗ‘з”ЁжҲ·жҠҘе‘Ҡз»“жһң
    • жҸҗдҫӣз”ұдёҖдёӘжҲ–еӨҡдёӘ TestEngine е®һзҺ°зҡ„пјҢз”ЁдәҺжү§иЎҢжөӢиҜ•зҡ„ API
    • йҖҡиҝҮ IDE е’Ңжһ„е»әе·Ҙе…·еҸ‘иө·жөӢиҜ•жү§иЎҢе·ҘдҪң
    • Launcher з»„д»¶иҙҹиҙЈжү§иЎҢеңЁжөӢиҜ•еҸ‘зҺ°жңҹй—ҙеҲӣе»әзҡ„жөӢиҜ•и®ЎеҲ’
еҗҺеҗ‘е…је®№жҖ§пјҡJUnit Vintage

JUnit Vintage еҸҜзЎ®дҝқзҺ°жңү JUnit жөӢиҜ•иғҪдёҺдҪҝз”Ё JUnit Jupiter еҲӣе»әзҡ„ж–°жөӢиҜ•дёҖеҗҢиҝҗиЎҢгҖӮ

JUnit Vintage жң¬иә«з”ұдёӨдёӘжЁЎеқ—з»„жҲҗпјҡ

  • junitпјҡjunit жҳҜз”ЁдәҺ JUnit 3 е’Ң JUnit 4 зҡ„ APIгҖӮ
  • junit-vintage-engineпјҡжҳҜеңЁ JUnit Platform дёҠиҝҗиЎҢ JUnit 3 е’Ң JUnit 4 жөӢиҜ•зҡ„жөӢиҜ•еј•ж“ҺгҖӮ

2. JUnit Jupiter зј–еҶҷ

жіЁи§Ј

JUnit 4 дёҺ JUnit 5 дёӯзҡ„еёёз”ЁжіЁи§ЈжҜ”иҫғ

JUnit 5 JUnit 4 иҜҙжҳҺ
@Test @Test иў«жіЁи§Јзҡ„ж–№жі•жҳҜдёҖдёӘжөӢиҜ•ж–№жі•гҖӮдёҺ JUnit 4 зӣёеҗҢгҖӮ
@BeforeAll @BeforeClass иў«жіЁи§Јзҡ„пјҲйқҷжҖҒпјүж–№жі•е°ҶеңЁеҪ“еүҚзұ»дёӯзҡ„жүҖжңү @Test ж–№жі•еүҚжү§иЎҢдёҖж¬ЎгҖӮ
@AfterAll @AfterClass иў«жіЁи§Јзҡ„пјҲйқҷжҖҒпјүж–№жі•е°ҶеңЁеҪ“еүҚзұ»дёӯзҡ„жүҖжңү @Test ж–№жі•еҗҺжү§иЎҢдёҖж¬ЎгҖӮ
@BeforeEach @Before иў«жіЁи§Јзҡ„ж–№жі•е°ҶеңЁеҪ“еүҚзұ»дёӯзҡ„жҜҸдёӘ @Test ж–№жі•еүҚжү§иЎҢгҖӮ
@AfterEach @After иў«жіЁи§Јзҡ„ж–№жі•е°ҶеңЁеҪ“еүҚзұ»дёӯзҡ„жҜҸдёӘ @Test ж–№жі•еҗҺжү§иЎҢгҖӮ
@Disabled @Ignore иў«жіЁи§Јзҡ„ж–№жі•дёҚдјҡжү§иЎҢпјҲе°Ҷиў«и·іиҝҮпјүпјҢдҪҶдјҡжҠҘе‘Ҡдёәе·Іжү§иЎҢгҖӮ
@ExtendWith @RunWith ж”ҫеңЁжөӢиҜ•зұ»еҗҚд№ӢеүҚпјҢз”ЁжқҘзЎ®е®ҡиҝҷдёӘзұ»жҖҺд№ҲиҝҗиЎҢзҡ„
@ExtendWith @Rule дёҖз»„е®һзҺ°дәҶTestRuleжҺҘеҸЈзҡ„е…ұдә«зұ»
@ExtendWith @ClassRule з”ЁдәҺжөӢиҜ•зұ»дёӯзҡ„йқҷжҖҒеҸҳйҮҸпјҢеҝ…йЎ»жҳҜTestRuleжҺҘеҸЈзҡ„publicе®һдҫӢ
@Tag @Category иў«з”ЁдәҺйҖҡиҝҮеЈ°жҳҺж ҮзӯҫжқҘиҝҮж»ӨжөӢиҜ•ж–№жі•
@TestFactory еЈ°жҳҺиҝҷдёӘж–№жі•жҳҜй’ҲеҜ№дәҺdynamic testsжөӢиҜ•е·ҘеҺӮ
@DisplayName з»ҷиҝҷдёӘзұ»жҲ–иҖ…ж–№жі•и®ҫе®ҡдёҖдёӘзү№ж®Ҡзҡ„еҗҚеӯ—
@Nested еЈ°жҳҺиҝҷдёӘж–№жі•жҳҜ дёҖдёӘеөҢеҘ—зҡ„, йқһйқҷжҖҒзҡ„ж–№жі•
дҪҝз”ЁжіЁи§Ј

  }

еүҚзҪ®жқЎд»¶

еүҚзҪ®жқЎд»¶ (Assumption) дёҺж–ӯиЁҖзұ»дјјпјҢдҪҶеүҚзҪ®жқЎд»¶еҝ…йЎ»дёә trueпјҢеҗҰеҲҷжөӢиҜ•е°ҶдёӯжӯўгҖӮ

public void testAdd_OnlyOnFriday() {

  assumeTrue(ldt.getDayOfWeek().getValue() == 5);
  // Remainder of test (only executed if assumption holds)...
}
assumingThat()

ж— и®ә assumingThat() дёӯзҡ„еүҚзҪ®жқЎд»¶жҲҗз«ӢдёҺеҗҰпјҢйғҪдјҡжү§иЎҢ lambda иЎЁиҫҫејҸеҗҺзҡ„жүҖжңүд»Јз Ғ

@Test
@DisplayName("This test is only run on Fridays (with lambda)")
public void testAdd_OnlyOnFriday_WithLambda() {
  LocalDateTime ldt = LocalDateTime.now();
  assumingThat(ldt.getDayOfWeek().getValue() == 5,
      () -> {
        // Execute this if assumption holds...
      });
  // Execute this regardless
}

дёҖдёӘжөӢиҜ•зұ»еҸҜеЈ°жҳҺеӨҡдёӘ @RuleпјҢиҝҷдәӣи§„еҲҷеҸҜеңЁжөӢиҜ•ж–№жі•зә§еҲ«е’Ңзұ»зә§еҲ«дёҠиҝҗиЎҢ

жү©еұ•зӮ№е’ҢжөӢиҜ•з”ҹе‘Ҫе‘Ёжңҹ

дёҖдёӘжү©еұ•зӮ№еҜ№еә”дәҺ JUnit test з”ҹе‘Ҫе‘ЁжңҹдёӯдёҖдёӘйў„е®ҡд№үзҡ„зӮ№гҖӮ

жҺҘеҸЈ иҜҙжҳҺ
AfterAllCallback е®ҡд№ү API жү©еұ•пјҢеёҢжңӣеңЁи°ғз”ЁжүҖжңүжөӢиҜ•еҗҺи®©жөӢиҜ•е®№еҷЁжү§иЎҢйўқеӨ–зҡ„иЎҢдёәгҖӮ
AfterEachCallback е®ҡд№ү API жү©еұ•пјҢеёҢжңӣеңЁи°ғз”ЁжҜҸдёӘжөӢиҜ•ж–№жі•еҗҺи®©жөӢиҜ•жү§иЎҢйўқеӨ–зҡ„иЎҢдёәгҖӮ
AfterTestExecutionCallback е®ҡд№ү API жү©еұ•пјҢеёҢжңӣеңЁжү§иЎҢжҜҸдёӘжөӢиҜ•еҗҺи®©жөӢиҜ•з«ӢеҚіжү§иЎҢйўқеӨ–зҡ„иЎҢдёәгҖӮ
BeforeAllCallback е®ҡд№ү API жү©еұ•пјҢеёҢжңӣеңЁи°ғз”ЁжүҖжңүжөӢиҜ•еүҚи®©жөӢиҜ•е®№еҷЁжү§иЎҢйўқеӨ–зҡ„иЎҢдёәгҖӮ
BeforeEachCallback е®ҡд№ү API жү©еұ•пјҢеёҢжңӣеңЁи°ғз”ЁжҜҸдёӘжөӢиҜ•еүҚи®©жөӢиҜ•жү§иЎҢйўқеӨ–зҡ„иЎҢдёәгҖӮ
BeforeTestExecutionCallback е®ҡд№ү API жү©еұ•пјҢеёҢжңӣеңЁжү§иЎҢжҜҸдёӘжөӢиҜ•еүҚи®©жөӢиҜ•з«ӢеҚіжү§иЎҢйўқеӨ–зҡ„иЎҢдёәгҖӮ
ParameterResolver е®ҡд№ү API жү©еұ•пјҢеёҢжңӣеңЁиҝҗиЎҢж—¶еҠЁжҖҒи§ЈжһҗеҸӮж•°гҖӮ

public class GeneratedPersonParameterResolver implements ParameterResolver {

  @Override
  public boolean supportsParameter(ParameterContext parameterContext, ExtensionContext extensionContext)
      throws ParameterResolutionException {
    return parameterContext.getParameter().getType() == Person.class;
  }

  @Override
  public Object resolveParameter(ParameterContext parameterContext, ExtensionContext extensionContext)
      throws ParameterResolutionException {
    return PersonGenerator.createPerson();
  }

}
