1.2 MySQL иҜҰз»Ҷе®үиЈ…жӯҘйӘӨ

иҝҗиЎҢ mysql-installer-community-5.7.28.0.msiпјҢиҝӣе…Ҙе®үиЈ…жӯҘйӘӨ

еҗҢж„ҸеҚҸи®®пјҢзӮ№еҮ»дёӢдёҖжӯҘ
йҖүжӢ©е®үиЈ…зұ»еһӢпјҢзӮ№еҮ»дёӢдёҖжӯҘ
жіЁж„Ҹпјҡе®үиЈ…йңҖиҰҒдҫқиө–зҺҜеўғпјҢеҰӮйңҖе®үиЈ…зҡ„еҠҹиғҪзјәе°‘еҜ№еә”зҺҜеўғпјҢйңҖе…Ҳе®үиЈ…зҺҜеўғ
дёӢдёҖжӯҘзҡ„ж—¶еҖҷдјҡжҸҗзӨәзјәе°‘зҺҜеўғ,жӯӨеӨ„зҡ„жҸҗзӨәжҳҜиҜҙдҪ жңҹжңӣе®үиЈ…зҡ„дә§е“Ғдёӯзјәе°‘дҫқиө–,дёҚдёҖе®ҡжҸҗзӨәзҡ„жҳҜж•°жҚ®еә“жңҚеҠЎеҷЁе®үиЈ…зҡ„дҫқиө–,еҸҜиғҪжҳҜе…¶д»–еҠҹиғҪз»„д»¶зҡ„дҫқиө–зјәе°‘
жӯӨеӨ„еҸ‘зҺ°жІЎжңү mysql server, иҜҙжҳҺе®ғеҜ№еә”зҡ„дҫқиө–зјәе°‘,еҸҜд»Ҙе…Ҳжү§иЎҢе®үиЈ…иҝҷдәӣеҸҜд»Ҙе®үиЈ…зҡ„,д№ӢеҗҺе®үиЈ…е®Ңдҫқиө–еҗҺеҶҚйҮҚж–°е®үиЈ…е…¶д»–жЁЎеқ—,жҲ‘д»¬жӯӨеӨ„зҡ„ж“ҚдҪңжҳҜе…Ҳе®үиЈ…дҫқиө–,з„¶еҗҺеҗҺйҖҖеҶҚж¬Ўе®үиЈ…
дёӢиҪҪе®үиЈ…еүҚйқўжҸҗзӨәзҡ„ mysql server йңҖиҰҒзҡ„ vc2013еҗҺ,vc2013 еҰӮжһңеңЁ64дҪҚзі»з»ҹдёҠйқўе»әи®®32е’Ң64зҡ„зЁӢеәҸйғҪе®үиЈ…,з„¶еҗҺеңЁдёҠеӣҫзҡ„ең°ж–№еҗҺйҖҖ,йҮҚж–° next е°ұеҸҜд»ҘзңӢеҲ°еҸҜд»Ҙе®үиЈ… mysql server дәҶ

1.3 е…ій—ӯејҖжңәиҮӘеҗҜгҖҒжүӢеҠЁеҗҜеҠЁе‘Ҫд»Ө

3. MySQL еҹәжң¬е‘Ҫд»Ө

жҹҘзңӢMySQLдёӯжүҖжңүж•°жҚ®еә“

#иҝһжҺҘMySQL
mysql> SHOW DATABASES;  #жҳҫзӨәеҪ“еүҚMySQLдёӯеҢ…еҗ«зҡ„жүҖжңүж•°жҚ®еә“

еҲӣе»әиҮӘе®ҡд№үж•°жҚ®еә“: CREATE DATABASE

mysql> CREATE DATABASE mydb1;  #еҲӣе»әmydbж•°жҚ®еә“
mysql> CREATE DATABASE mydb2 CHARACTER SET gbk;  #еҲӣе»әж•°жҚ®еә“е№¶и®ҫзҪ®зј–з Ғж јејҸgbkгҖҗжҺЁиҚҗгҖ‘
mysql> CREATE DATABASE mydb3 CHARACTER SET gbk COLLATE gbk_chinese_ci;  #ж”ҜжҢҒз®ҖдҪ“дёӯж–Үе’Ңз№ҒдҪ“дёӯж–Ү
mysql> CREATE DATABASE IF NOT EXISTS mydb4;  #еҰӮжһңmydb4ж•°жҚ®еә“дёҚеӯҳеңЁпјҢеҲҷеҲӣе»әпјӣеҰӮжһңеӯҳеңЁпјҢеҲҷдёҚеҲӣе»әгҖӮ

жҹҘзңӢж•°жҚ®еә“еҲӣе»әдҝЎжҒҜ: SHOW CREATE DATABASE

mysql> SHOW CREATE DATABASE mydb2; #жҹҘзңӢеҲӣе»әж•°жҚ®еә“ж—¶зҡ„еҹәжң¬дҝЎжҒҜ

дҝ®ж”№ж•°жҚ®еә“: ALTER DATABASE

mysql> ALTER DATABASE mydb2 CHARACTER SET gbk; #жҹҘзңӢеҲӣе»әж•°жҚ®еә“ж—¶зҡ„еҹәжң¬дҝЎжҒҜ
  • жіЁж„ҸпјҡmysqlдёӯUTF-8зҡ„еӯ—з¬ҰйӣҶдёә utf8

еҲ йҷӨж•°жҚ®еә“: DROP DATABASE

mysql> DROP DATABASE mydb1; #еҲ йҷӨж•°жҚ®еә“mydb1

жҹҘзңӢеҪ“еүҚжүҖдҪҝз”Ёзҡ„ж•°жҚ®еә“ SELECT DATABASE()

mysql> SELECT DATABASE(); #жҹҘзңӢеҪ“еүҚдҪҝз”Ёзҡ„ж•°жҚ®еә“

дҪҝз”Ёж•°жҚ®еә“ USE

mysql> USE mydb1; #дҪҝз”Ёmydb1ж•°жҚ®еә“
