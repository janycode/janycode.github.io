● 运行容器

注意事项：

  • 运行容器需要指定具体的镜像，如果镜像不存在，会直接下载。
  • WARNING: IPv4 forwarding is disabled. Networking will not work.
    # echo "net.ipv4.ip_forward=1" >>/usr/lib/sysctl.d/00-system.conf
    # systemctl restart network && systemctl restart docker
    再次 docker run … 即可解决。
