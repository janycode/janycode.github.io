09-JSP 分页查询

1. 分页查询准备工作

采用物理查询：页面查询一页，就从数据库里查询一页数量的数据。
优：减少单次查询数据库的时间
缺：增加了操作数据库的次数

所需的关于 页数 的数据：
请求：当前页数 currentPage
响应：PageBean 类封装
● 当前页数 currentPage
● 总页数 totalPage
● 总记录数 totalSize
● 每页记录数 pageSize
● 当前页数据 pageList

总页数 = (总记录数 % 每页记录数 == 0) ? (总记录数 / 每页记录数) : (总记录数 / 每页记录数 + 1);
当前页数据 list = query( select * from limit 每页记录数*(当前页数-1), 每页记录数 );
