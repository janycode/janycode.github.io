## PhotoShop

UI作图，前端测量、吸取颜色、切图。



> PhotoShop绿色版：https://pan.quark.cn/s/35e0b2cbe809
>
> 解决2K、4K等高分屏下Photoshop窗口、字体太小等问题：https://blog.csdn.net/wenzhongxiang/article/details/128815451



拿到设计稿后：使用ps打开

1. 图片上面右键，打开方式选择photoshop
2. ps里 文件 ->打开

图片放大和缩小：

1. Ctrl + +
2. Ctrl + -
3. Alt + 滚动

图片的移动：

1. 按住空格，鼠标变小手，拖动图片

如何调整出来标尺：

1. Ctrl + R
2. 作用：拖动参考线方便测量
3. 视图里找到标尺，把对勾勾选

测量图片大小：

1. 使用矩形选框工具（左侧竖着第二个）
2. 如何查看数据大小（窗口 -> 信息面板 -> W:宽 H:高）
3. 如何修改测量单位：在标尺上面右键取修改单位 --> 像素
4. Ctrl + D 取消选区
5. 选完后想调整大小，可以右键 -> 变换选区

吸取颜色：

1. 使用吸管工具
2. 吸取颜色完成后，点击左下角的背景色，会有弹窗，在弹窗里有#十六禁止的颜色值可以复制

切图：

1. 使用快捷键截图，每次只能截取一个
   使用选框工具截取区域：Ctrl + C >> Ctrl + N >> 回车 >> Ctrl + V >> Ctrl + S >> 回车 >> 回车
2. 切片工具（裁剪工具进行切换）
   使用切片工具框选要留住的区域，点击【文件】-【存储为web所用格式】，弹窗里面点击存储。
   弹窗 >> 格式：仅限图像，切片：所有用户切片。
