### 1. 图片随网页大小变化

超过屏幕宽度时，缩小，高度等比例缩小，小于屏幕宽度时，显示原图尺寸：

```css
img{
	height: auto; 
	max-width: 100%;
}
```

始终随容器宽度变化：

```css
img{
	width: 100%;
	height: auto;
}
```

### 2. 网页整体内容居中

```css
/* 如果居中后还有留白，则可能内部标签是有 padding 值 */
body{
	width: 100%;
	margin-left: auto;
	margin-right: auto;
}
```

```css
/* 需要两边留白且居中的主体内容，将主体内容的宽度设置为80% */
.content {
    width: 80%;
}
```
